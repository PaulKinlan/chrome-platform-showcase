IDB Large Value Error

IndexedDB reports detailed errors for large-value read failures instead of a generic UNKNOWN_ERROR. Debuggable now.

The canonical shape of the API:

// IDBRequest.onerror now receives detailed error info
// for large-value read failures (UNKNOWN_ERROR was
// the previous useless placeholder).
