Scope Budget Visualizer

Chrome 130 adds attribution_scopes to the Attribution Reporting API. A source can carry up to 20 scopes; only triggers that share a scope are eligible to attribute to that source. This visualizer lets you assign scopes to a source, fire conversion triggers with different scope values, and watch the attribution budget meter — plus see exactly which triggers succeed or are filtered out.

// Chrome 130: attribution_scopes on a source registration
// Response to Attribution-Reporting-Register-Source request:
{
  "destination": "https://advertiser.example",
  "source_event_id": "12345",
  "expiry": 2592000,

  "attribution_scopes": {
    "values":      ["campaign-a", "brand-video", "retargeting"],
    "limit":       3,          // max active scopes at once
    "max_event_states": 3      // independent event counters per scope
  }
}

// Trigger registration only matches if scope overlaps:
// Attribution-Reporting-Register-Trigger: { "attribution_scopes": ["campaign-a"] }
// → attributed ✓

// Attribution-Reporting-Register-Trigger: { "attribution_scopes": ["campaign-b"] }
// → filtered ✗ (no overlap)
