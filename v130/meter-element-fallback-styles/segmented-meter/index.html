Segmented Meter

Three distinct meter designs — battery bars, audio VU levels, and rating stars — all built from a single CSS technique: layering a repeating-linear-gradient gap over a <meter appearance: none> fill. Chrome 130's authored fallback styles make this possible.

the code

/* The repeating-gradient gap overlay technique */

/* 1. Remove UA appearance */
meter {
  appearance: none;
  background: var(--bg-stone);
}

/* 2. Style the fill */
meter::-webkit-meter-optimum-value {
  background: var(--accent-emerald);
}

/* 3. Overlay gap lines on the wrapper */
.meter-wrap::after {
  content: '';
  position: absolute;
  inset: 0;
  background: repeating-linear-gradient(
    to right,
    transparent 0,
    transparent calc((100% / 10) - 2px), /* segment width minus gap */
    var(--bg-paper) calc((100% / 10) - 2px),
    var(--bg-paper) calc(100% / 10)
  );
  pointer-events: none;
}
