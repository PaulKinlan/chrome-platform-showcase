Synthetic PointerEvent targets

Click "fire synthetic pointermove" to dispatch a JS-constructed PointerEvent that carries a list of coalesced sub-events, each targeted at a different box. Before v130, all sub-events were retargeted to the parent — now they keep the targets you set.

the code

const a = document.getElementById("a");
const b = document.getElementById("b");

const coalesced1 = new PointerEvent("pointermove", { pointerId: 1 });
Object.defineProperty(coalesced1, "target", { value: a });

const coalesced2 = new PointerEvent("pointermove", { pointerId: 1 });
Object.defineProperty(coalesced2, "target", { value: b });

const ev = new PointerEvent("pointermove", {
  pointerId: 1,
  coalescedEvents: [coalesced1, coalesced2],
});

a.dispatchEvent(ev);

// In v130+: ev.getCoalescedEvents()[0].target === a
//           ev.getCoalescedEvents()[1].target === b
// Pre-v130:  both retargeted to the parent event's target.
