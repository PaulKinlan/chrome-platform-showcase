Modify Metadata

Encoded Transform gets setMetadata helpers. Pipelines can mutate per-frame metadata cleanly without rebuilding the frame.

The canonical shape of the API:

transformer.transform = (frame, ctl) => {
  frame.setMetadata({ ...frame.getMetadata(), payloadType: 96 });
  ctl.enqueue(frame);
};
