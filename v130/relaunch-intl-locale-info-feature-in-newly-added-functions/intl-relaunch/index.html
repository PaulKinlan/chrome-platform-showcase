Intl Locale Info relaunch

Intl Locale Info features re-introduced through explicit method calls (getCalendars, getHourCycles, etc.) instead of the old getter properties.

The canonical shape of the API:

const l = new Intl.Locale("en");
console.log(l.getCalendars(), l.getHourCycles());
