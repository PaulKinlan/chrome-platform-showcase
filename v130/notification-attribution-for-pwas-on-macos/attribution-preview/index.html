Attribution before & after

Attribution is Notification-Center UI — a Linux or Windows browser cannot observe it, and a JS probe that claimed to would be lying. So this page does two honest things: a rendered before/after comparison of what macOS users see, and a real environment probe of the pieces that decide which rendering you'd get: platform, installed-PWA display mode, and notification permission.

Manual observation, not a live probe. The two cards below are rendered mock-ups of macOS Notification Center. To see the real thing: install this site (or any PWA) on macOS with Chrome 130+ in the developer trial (rolling out through Chrome 152), grant notifications, and fire one from the tester.
