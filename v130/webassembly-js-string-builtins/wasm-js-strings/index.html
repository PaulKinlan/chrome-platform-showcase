Wasm JS String Builtins

Wasm gains imported JS string built-ins. Cross-boundary string ops (length, charCodeAt, equality) inline into native code.

The canonical shape of the API:

// Wasm modules can declare imports against the
// `wasm:js-string` namespace and get specialised
// inlined string operations.
