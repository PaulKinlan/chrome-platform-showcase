Pseudos after ::part()

::part() composition now accepts more pseudo-classes and pseudo-elements. Shadow-DOM design systems get richer styling hooks.

The canonical shape of the API:

my-button::part(button):hover { background: #eef; }
my-button::part(button)::before { content: "★"; }
