Part State Machine

Chrome 130 allows pseudo-classes like :hover, :focus, :disabled and pseudo-elements like ::before, ::after, ::placeholder after ::part(). This means custom element host pages can now style internal parts based on their interactive states — without the shadow DOM needing to expose JavaScript APIs or fire custom events.

What Chrome 130 enables

::part(btn):hover
Before: JS listener + data-hover attribute workaround
Chrome 130: direct CSS selector ✓
Style hover state of a shadow part directly from the host document's stylesheet.
::part(btn):focus
Before: shadow root must expose :focus styling
Chrome 130: host can add focus ring ✓
Host page can inject a custom focus indicator without modifying the component.
::part(input)::placeholder
Before: impossible to style from outside shadow
Chrome 130: full placeholder styling ✓
Style placeholder text colour, font, opacity from the host stylesheet.
::part(icon)::before
Before: no generated content injection
Chrome 130: inject content via CSS ✓
Add decorative ::before / ::after content to shadow parts from outside.
::part(btn):disabled
Before: must observe disabled attribute from JS
Chrome 130: CSS-only disabled style ✓
Apply dimmed/greyed appearance when a shadow input or button is disabled.
::part(check):checked
Before: no cross-shadow checked styling
Chrome 130: checked state from host ✓
Style a checked checkbox or radio inside shadow DOM from host CSS.
/* Chrome 130: pseudo-classes and pseudo-elements after ::part() */

/* Hover state */
my-button::part(button):hover {
  background: rgba(59, 130, 246, 0.25);
  border-color: #2563eb;
}

/* Focus ring injected from host page */
my-input::part(input):focus {
  outline: 3px solid #3b82f6;
  outline-offset: 2px;
}

/* Generated content added to a shadow part */
my-badge::part(icon)::before {
  content: "★ ";
  color: gold;
}

/* Placeholder styling of a shadow input */
my-field::part(input)::placeholder {
  color: var(--text-muted);
  font-style: italic;
}

/* Disabled appearance */
my-form::part(submit):disabled {
  opacity: 0.4;
  cursor: not-allowed;
}
