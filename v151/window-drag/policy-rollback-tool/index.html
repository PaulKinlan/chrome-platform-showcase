Policy & Rollback Tool

Three generators in one page: a manifest snippet builder for Window Controls Overlay, a CSS migration converter from -webkit-app-region to window-drag, and an analytics continuity checker that shows how to track drag-region interactions without breaking existing event listeners.

1 — Manifest snippet builder

Fill in app details to generate a manifest snippet with Window Controls Overlay enabled. WCO is required for full window-drag titlebar control.

2 — CSS migration converter

Paste CSS that uses -webkit-app-region and choose an output strategy.

3 — Analytics continuity checker

Drag regions swallow pointer events — existing click / analytics listeners on those elements silently stop firing. This checker probes your current environment and generates a safe wrapper pattern.
