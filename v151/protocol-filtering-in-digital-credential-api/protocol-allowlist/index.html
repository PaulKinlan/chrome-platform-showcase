Protocol allowlist checker

DigitalCredential.userAgentAllowsProtocol() is the real, synchronous way to ask the browser whether it will broker a given exchange protocol — before you build a request around it.

How it works

if ("DigitalCredential" in window &&
    typeof DigitalCredential.userAgentAllowsProtocol === "function") {
  DigitalCredential.userAgentAllowsProtocol("openid4vp");  // true
  DigitalCredential.userAgentAllowsProtocol("my-own-proto"); // false once filtering is on
}
