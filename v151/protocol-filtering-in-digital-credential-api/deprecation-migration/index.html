Deprecation & migration guide

Arbitrary opaque protocols are on the clock: deprecated from Chrome 151, gone in Chrome 160. Check whether your protocol survives, and copy the compliant pattern to migrate to.

Checking filtering state…
Chrome 151
Filtering begins. Unspecified presentation & issuance protocols are deprecated. userAgentAllowsProtocol() ships.
151 → 159
Transition window. Migrate opaque-pipeline requests to the normative protocol set.
Chrome 160
Arbitrary protocols removed. Only normatively-referenced protocols are brokered.
