Prose Comparator

The same paragraph rendered with ruby-overhang: auto and ruby-overhang: none, side-by-side, for Japanese, Chinese pinyin, and Korean transliteration. Toggle a character-cell ruler to see exactly where the annotations overflow their base.

Requires a Chrome 151+ build. If your browser does not yet support the property, both panels render the same way (browser default).

language

compare

ruby-overhang: auto (default, 151+)

Annotations may extend past their base character to reduce gaps in the running text.

ruby-overhang: none

Annotations are clipped to their base — wider tracking, no overhang into neighbouring characters.

code

.prose      { ruby-overhang: auto; }   /* default in Chrome 151 */
.legal-doc  { ruby-overhang: none; }   /* legal / mono layouts —
                                          predictable grid wins
                                          over readable flow */
