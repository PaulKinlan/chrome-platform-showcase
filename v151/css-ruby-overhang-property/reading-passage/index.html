values explained

  • auto (default) — the browser applies established CJK typographic practice: overhang into adjacent whitespace and CJK punctuation is allowed. Most natural for general text.
  • spaces — ruby may overhang into adjacent whitespace characters (U+0020, ideographic space) but not CJK punctuation marks.
  • none — no overhang allowed. The line is padded symmetrically around each base character to fit the annotation, which can produce noticeable gaps in dense text.

code

/* Default: allow overhang into whitespace and CJK punctuation */
ruby { ruby-overhang: auto; }

/* Only overhang into whitespace, not punctuation */
ruby { ruby-overhang: spaces; }

/* No overhang — line expands symmetrically to fit annotations */
ruby { ruby-overhang: none; }

/* Typical prose setup */
.japanese-body {
  ruby-overhang: auto;
  line-height: 2.4; /* extra space for ruby annotations */
}
