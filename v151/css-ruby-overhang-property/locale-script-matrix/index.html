Vertical writing

In vertical writing mode (writing-mode: vertical-rl), overhang applies in the block direction. The same property controls whether ruby text overhangs into adjacent block-direction space.

CSS property values
auto — browser applies its default typographic rules. Ruby may overhang adjacent spaces or CJK punctuation that has natural half-width space. Matches the established convention in print Japanese typography.
spaces — ruby may only overhang adjacent space characters and CJK punctuation with inherent space. Does not overhang other content.
none — no overhang allowed. The ruby annotation is contained within the bounds of its base text. Any excess annotation width forces the base text to expand, leaving visible gaps in the line.
