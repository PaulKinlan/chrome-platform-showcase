Feature Detection — form_submission

Check whether the browser supports speculation rules, whether form_submission is accepted in a rule set, and inspect any active prerendering state.

The form_submission field (Chrome 151) extends speculation rules to support prerendering form submissions — not just link navigations. The probe injects a real <script type="speculationrules"> with a form_submission entry and checks if the browser accepts it.
