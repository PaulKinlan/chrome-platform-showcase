how it works

The from keyword in a color function (e.g. rgb(), hsl(), oklch()) lets you reference an origin color's channels using the channel variables r, g, b, alpha, etc. For relative alpha colors, you keep the color channels unchanged and only modify the alpha channel — typically with calc() to multiply or set a new value.
