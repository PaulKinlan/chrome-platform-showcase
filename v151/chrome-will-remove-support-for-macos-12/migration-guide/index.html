Migration Guide

Hardware compatibility, macOS upgrade paths, and enterprise fleet management guidance for the Chrome 151 removal of macOS 12 Monterey support.

Probe has not run yet.
Users who remain on macOS 12 after Chrome 151 ships will not receive further Chrome security updates. Staying current on Chrome requires upgrading to macOS 13 (Ventura) or later.

hardware compatibility

Mac model Runs macOS 13 Ventura? Action required
MacBook Air / Pro (2018 or later) Yes Upgrade to macOS 13+ via System Settings → General → Software Update
Mac mini (2018 or later) Yes Upgrade via Software Update
Mac Pro (2019 or later) Yes Upgrade via Software Update
iMac (2019 or later) Yes Upgrade via Software Update
MacBook Air (2017 or earlier) No Hardware does not support macOS 13. Consider device replacement or use an alternative browser that still supports older macOS.
MacBook Pro (2017 or earlier) No Hardware does not support macOS 13. Same as above.
