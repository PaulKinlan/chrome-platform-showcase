Duplicate Navigation Demo

Rapidly click the same "navigate" button (which sets location.hash). In Chrome 151+, duplicate identical navigations are silently ignored instead of restarting the navigation cycle. The log shows which clicks actually triggered a navigation event.

Navigation events log

In Chrome 151+, clicking "#same" rapidly should produce fewer hashchange events than clicks — duplicates are deduplicated.
