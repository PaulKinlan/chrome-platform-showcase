aria-actions

A new ARIA attribute that exposes secondary action buttons within composite interactive widgets for improved screen-reader discoverability. Instead of nesting actions inside the accessible name, aria-actions provides a dedicated reference list.

concepts

  1. Action Card Grid

    A grid of cards, each with a primary label and secondary action buttons (edit, delete, share) linked via aria-actions. Inspect the accessibility tree to see how actions are exposed.

  2. Tree View Actions

    A file-tree widget where each node has expand/collapse plus secondary actions (rename, move). aria-actions connects them for assistive tech without polluting the node's accessible name.
