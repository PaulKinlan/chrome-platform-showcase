Timing API Probe

Fetch a cross-origin resource (a public image redirect). Then inspect the PerformanceResourceTiming entry. Cross-origin redirect timing fields are zeroed unless the redirect server sends Timing-Allow-Origin.

FieldValueStatus

Zeroed fields (0) indicate the timing data is hidden due to cross-origin restrictions. The Chrome 151 opt-in allows redirect servers to expose this data.
