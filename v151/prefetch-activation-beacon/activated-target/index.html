Activation target

This is the page that gets prefetched. When you arrive, it inspects its own navigation entry to tell you whether it was served from the prefetch cache.

Reading navigation timing…

The activation beacon fires precisely when deliveryType is "navigational-prefetch": the browser committed a navigation using a prefetched response. That is the reliable signal on-prefetch-activation reports to the server — no page script required.
