Density Explorer

Three scrollers side by side — one each for scrollbar-width: auto, thin, and none. Sweep a colour pair for scrollbar-color, layer in a Fluent-style override, and read the computed style each one resolves to so you can decide where to draw the line between Fluent default and a custom skin.

Note: Fluent scrollbar styling ships on Chrome / Windows. macOS / Linux render their own native scrollbar style; the property values are still inspectable.
