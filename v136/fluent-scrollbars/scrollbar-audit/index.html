Chrome 136 on Windows/Linux renders Fluent-style scrollbars by default. Author CSS using standard scrollbar-color and scrollbar-width properties always wins. The deprecated ::-webkit-scrollbar pseudo-elements also work but may produce inconsistent results when mixed with the Fluent theme. This audit shows which approach each rule uses.
