what we're looking for

  • any <h1> nested inside sectioning content (article, section, aside, nav) — pre-136 the UA shrunk it
  • headings that relied on the shrink — without an author font-size, they'll now render at the document-level 2em
  • recommended fix: explicit font-size in your author CSS for each h1 inside sectioning roots
