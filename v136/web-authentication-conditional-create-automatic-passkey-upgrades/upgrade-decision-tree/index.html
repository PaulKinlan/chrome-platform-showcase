Upgrade Decision Tree

Conditional create only succeeds when the user agent considers it safe to silently mint a passkey. The tree below walks every input signal — signed-in user, recent password autofill, hardware availability, RP exclude list — and shows whether each scenario triggers an upgrade, a fallback prompt, or a no-op.

Note: conditional create ships in Chrome 136 with a platform authenticator. Without a hardware roaming-or-platform authenticator the request will resolve with no credential.
