Remote Cursor

Mouse + key passthrough into a captured tab. Clicks on the preview tile route to controller.forwardClick(), key strokes to controller.forwardKey(). A ghost cursor follows the local pointer so the remote user can see where the local one is.

Note: Captured Surface Control is behind chrome://flags/#captured-surface-control in Chrome 136. The ghost cursor still tracks locally without the flag.
