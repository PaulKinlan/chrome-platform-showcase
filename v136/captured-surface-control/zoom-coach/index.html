Zoom Coach

Captured Surface Control's third leg: programmatic zoom of the captured tab. Read getSupportedZoomLevels() for the discrete set the browser allows, snap to the closest level on a slider, and watch setZoomLevel() drive the captured surface up or down.

Note: Captured Surface Control is behind chrome://flags/#captured-surface-control in Chrome 136. The page still loads but zoom calls will throw until the flag is enabled and a tab is captured.

    the API

    const stream = await navigator.mediaDevices.getDisplayMedia({
  video: true,
  preferCurrentTab: false,
});
const track = stream.getVideoTracks()[0];

// 1. Discrete set the user agent supports.
const levels = track.getSupportedZoomLevels();   // e.g. [25, 33, 50, 67, 75, 90, 100, 110, 125, 150, ...]

// 2. Snap to a level. setZoomLevel rejects if the level is not in the list.
await track.setZoomLevel(150);

// 3. Read back.
const settings = track.getSettings();
console.log(settings.zoomLevel); // 150
