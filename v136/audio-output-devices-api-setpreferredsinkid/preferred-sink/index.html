setPreferredSinkId

setPreferredSinkId persists a user's chosen audio output for the origin. Saves the per-session re-selection that setSinkId required.

The canonical shape of the API:

await audio.setPreferredSinkId(deviceId);
// Same as setSinkId but persists across reloads as
// the user's preference for this origin.
