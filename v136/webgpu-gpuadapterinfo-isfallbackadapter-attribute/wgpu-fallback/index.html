isFallbackAdapter

Request both a real adapter and a fallback-tier adapter, then inspect adapter.info.isFallbackAdapter on each. Sites can gate heavy rendering on a real GPU and fall back to a 2D canvas otherwise.

the code

const adapter = await navigator.gpu.requestAdapter();
if (adapter.info.isFallbackAdapter) {
  // CPU fallback — drop the fancy compute shader, render a placeholder.
} else {
  // Real GPU. Knock yourself out.
}
