Limit Mixer

Slide between standard, constrained-high, and high — or feed an arbitrary percentage to the new dynamic-range-limit-mix() function to interpolate between tiers. The preview's HDR overlay scales as you drag.

Note: dynamic-range-limit requires an HDR-capable display and the relevant flag. Without it the preview still shows the SDR pixels and the live style attribute reflects the property you chose.
