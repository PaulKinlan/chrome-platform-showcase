NVS in disk cache

No-Vary-Search rules now apply to the HTTP disk cache, not just memory. Long-tail tracking parameters no longer fragment the cache across sessions.

The canonical shape of the API:

# Response:
No-Vary-Search: params=("utm_source")
# /article?utm_source=email and /article are now
# the same disk-cache entry.
