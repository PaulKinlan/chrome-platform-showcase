Desktop Eager Tuning

Speculation rules' eager eagerness tunes its trigger threshold up on desktop, lifting prerender hit rate for sites where bandwidth is cheap.

The canonical shape of the API:

{ "prerender": [{ "where": { "and": [...] }, "eagerness": "eager" }] }
// Desktop "eager" now triggers more aggressively
// for likely-clicked targets, improving prerender
// hit rate.
