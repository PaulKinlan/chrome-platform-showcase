Custom Descriptors

Wasm Custom Descriptors standardise the way modules attach typed metadata to their interface. Tools no longer need to parse each toolchain's custom section format.

The canonical shape of the API:

// Wasm modules can attach typed metadata to imports,
// exports, and globals. Tools (debuggers, profilers)
// pick the descriptors up without parsing custom
// section conventions per-toolchain.
