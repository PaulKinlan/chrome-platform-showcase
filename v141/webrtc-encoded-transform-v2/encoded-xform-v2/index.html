Encoded Transform V2

WebRTC Encoded Transform V2 — refined streams + writer + metadata exposure. Crypto and ML pipelines can read/write encoded frames with much less ceremony than v1.

The canonical shape of the API:

const transformer = new RTCRtpScriptTransformer({
  transform(frame, ctl) {
    // Per-frame access with extra metadata in v2.
    ctl.enqueue(frame);
  }
});
sender.transform = transformer;
