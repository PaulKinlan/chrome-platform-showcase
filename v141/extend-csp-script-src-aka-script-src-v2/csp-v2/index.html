CSP script-src v2

script-src-v2 brings finer-grained source matching (URL patterns, integrity assertions, signature checks) on top of the existing CSP grammar.

The canonical shape of the API:

Content-Security-Policy:
  script-src-v2 \
    'self' \
    'integrity' 'matches'
# v2 supports per-source integrity hints and
# match patterns that v1's flat allowlist couldn't.
