Window Audio

getDisplayMedia gains a windowAudio capture mode — the picked window's audio only, no system-wide mix. Privacy and product clarity win.

The canonical shape of the API:

const stream = await navigator.mediaDevices.getDisplayMedia({
  audio: { windowAudio: true }
});
// Captures audio from the chosen window source
// without grabbing system-wide audio.
