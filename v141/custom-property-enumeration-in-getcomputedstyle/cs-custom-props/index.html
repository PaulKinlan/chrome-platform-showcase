Custom Property Enumeration

getComputedStyle now enumerates custom properties alongside standard ones. Lets tooling and design systems inspect the live token set without scraping CSS rules.

The canonical shape of the API:

const cs = getComputedStyle(el);
for (const name of cs) {
  if (name.startsWith("--")) console.log(name, cs.getPropertyValue(name));
}
