Type Specimen

A type specimen that sets letter-spacing and word-spacing as percentages. Because percentages scale with font-size, the same spacing values work proportionally across display, headline, subhead, body, and caption sizes — something impossible with fixed px spacing.

Chrome 145+ — letter-spacing: 5% and word-spacing: 10%. In older browsers percentage values are treated as invalid and fall back to 0, collapsing the spacing.

Drag the sliders to set percentage spacing · all sizes update proportionally · the CSS output shows the computed values at each font size

/* CSS percentage letter-spacing and word-spacing — Chrome 145 */

/* Before: fixed px means different visual weight at each font size */
.headline { font-size: 2rem; letter-spacing: 2px; }  /* 0.1em of size */
.body     { font-size: 1rem; letter-spacing: 2px; }  /* 0.2em of size — TOO WIDE */

/* After: percentage scales proportionally with font-size */
.headline { font-size: 2rem; letter-spacing: 5%; }  /* 1.6px */
.body     { font-size: 1rem; letter-spacing: 5%; }  /* 0.8px — stays proportional */

/* One value works at every size — no need for per-size overrides */
p { letter-spacing: 3%; word-spacing: 8%; }
