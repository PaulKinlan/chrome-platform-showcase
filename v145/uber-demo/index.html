Chrome 145 Uber Demo

Four ways into Chrome 145: two cohesive cross-feature applications, a complete themed release tour, and a live compatibility checker. The workshop and dashboard show features solving real product problems; the tour and checker make the unusually broad release easier to explore and diagnose.

concepts

  1. Typography Workshop

    A live reading environment built on eight Chrome 145 features: customizable <select> listbox, CSS percentage letter- and word-spacing, text-justify, column-wrap, clipboardchange, focus({ focusVisible }), overscroll on a non-root scroller, and Cookie Store maxAge for persistence.

  2. Performance Dashboard

    A working observability console wiring five Chrome 145 perf APIs: paintTime/presentationTime on performance entries, container-timing, bimodal performance (p50/p95), CSSPixels in LayoutShift, and the crash-reporting key/value API.

  3. v145 Tour

    Filterable, themed index of every concept demo across Chrome 145 — typography, security, payments, performance, platform. Quick way to skim the entire release from one page.

  4. Feature Compatibility Checker

    Live feature-detection probes for 18 Chrome 145 APIs — CSS percentage spacing, focus({ focusVisible }), WebGPU subgroups, JPEG XL, Cookie Store, crashReporter, IndexedDB, and more. Runs in your browser and shows exactly which features are supported, partial, or absent.

why four?

Chrome 145 shipped 30+ features across typography, observability, security, DevTools, and device APIs. Forcing all of them into one interface would produce a checklist rather than a believable product. The two integrated applications stay focused; the tour exposes every concept; and the compatibility checker reports what this browser can actually run.
