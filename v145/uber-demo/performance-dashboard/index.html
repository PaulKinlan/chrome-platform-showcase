Performance Dashboard

A real, working dashboard built on five Chrome 145 observability features: paint/presentation times on performance entries, container timing, bimodal performance timings (95th-percentile rendering), CSSPixels in LayoutShift, and the crash reporting key-value API. Drive it with the controls; watch the meters move.
