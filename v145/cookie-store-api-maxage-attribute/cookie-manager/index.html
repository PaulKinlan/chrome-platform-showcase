Cookie Manager

A live cookie manager built on the Cookie Store API. Set cookies with a maxAge slider (the Chrome 145 addition), watch them appear in the list with expiry bars, and see cookieStore.change events fire as cookies are created or deleted. Delete cookies with maxAge: 0.

Chrome 145+ — cookieStore.set({ maxAge }). In browsers without Cookie Store API support the form falls back to document.cookie and live events are not available.

Fill in a cookie name and value · pick a maxAge preset · click Set Cookie · watch the change event fire in the log · delete cookies with the × button

// Cookie Store API maxAge — Chrome 145
// Set cookies with relative duration instead of absolute timestamp.

// Chrome 145+ — use maxAge (seconds) — no clock dependency
await cookieStore.set({ name: 'session', value: 'abc', maxAge: 3600 }); // 1 hour

// Delete a cookie: maxAge: 0
await cookieStore.set({ name: 'session', value: '', maxAge: 0 });

// List all cookies
const all = await cookieStore.getAll();
all.forEach(c => console.log(c.name, c.expires));

// Listen for changes (create/delete events)
cookieStore.addEventListener('change', e => {
  console.log('changed:', e.changed.map(c => c.name));
  console.log('deleted:', e.deleted.map(c => c.name));
});
