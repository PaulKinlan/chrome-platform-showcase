Request Classifier

Type a URL. We resolve its address space and report whether Chrome would gate the request behind a permission prompt.

address spaces

  • loopback — 127.0.0.0/8, ::1 (most restricted)
  • private — 10/8, 172.16/12, 192.168/16, fd00::/8 (RFC 1918 + ULA)
  • public — everything else
