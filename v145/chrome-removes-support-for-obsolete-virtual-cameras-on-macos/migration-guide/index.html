vendor migration steps

// Virtual camera software vendors: migrate from DAL plug-in to camera extension
//
// 1. Register a Camera Extension in your macOS app:
//    - Target: com.apple.system-extension.camera
//    - Implement CMIOExtensionProvider protocol
//    - Deliver CMIOExtensionStream for video output
//
// 2. Request Camera Extension entitlement from Apple:
//    - com.apple.developer.system-extension.install
//    - Required for distribution outside TestFlight
//
// 3. Install the extension:
//    let manager = OSSystemExtensionManager.shared
//    manager.submitRequest(OSSystemExtensionRequest.activationRequest(
//        forExtensionWithIdentifier: "com.example.MyCameraExtension",
//        queue: .main
//    ))
//
// 4. Stream video:
//    - Push CMSampleBuffer frames to CMIOExtensionStream
//    - App appears as a standard camera in enumerateDevices()
