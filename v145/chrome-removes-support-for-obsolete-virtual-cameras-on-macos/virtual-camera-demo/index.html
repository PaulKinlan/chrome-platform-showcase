Virtual Camera Demo

Enumerates available video input devices to show which cameras Chrome 145 can access — useful for checking whether a virtual camera uses the modern macOS extension API.

device enumeration

mediaDevices API—
Platform—
Video inputs found—
Click “Enumerate cameras” to list devices.

code

const devices = await navigator.mediaDevices.enumerateDevices();
const cameras = devices.filter(d => d.kind === 'videoinput');
cameras.forEach(cam => {
  console.log(cam.label || '(label hidden until permission granted)');
  console.log('ID:', cam.deviceId);
});

// On macOS, labels are only revealed after getUserMedia() permission is granted.
// If a virtual camera that was visible in Chrome 144 is missing in Chrome 145,
// it used the obsolete CoreMediaIO DAL plug-in API.
