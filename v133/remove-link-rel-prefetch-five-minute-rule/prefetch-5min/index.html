Prefetch 5-min Rule Removal

Removes the legacy 5-minute survival heuristic for <link rel=prefetch>. Prefetched resources now follow standard Cache-Control semantics.

The canonical shape of the API:

# Prefetched resources no longer special-cased to
# 5-minute cache survival. Fall back to standard
# cache rules.
