:open

The :open pseudo matches <details>, <dialog>, and <select> when open. Single hook for the 'is this widget showing?' question.

The canonical shape of the API:

details:open { background: yellow; }
dialog:open { box-shadow: 0 8px 24px rgba(0,0,0,0.25); }
