setPreferredSinkId

Grant mic access, enumerate output devices, then route playback to the picked one via the per-element setSinkId and the page-level navigator.mediaDevices.setPreferredSinkId.

the code

// Page-level preference (new in Chrome 133):
await navigator.mediaDevices.setPreferredSinkId(deviceId);

// Per-element override (since Chrome 49):
await audioEl.setSinkId(deviceId);
