why the difference

Each step of an IndexedDB cursor is a separate task on the main thread. getAllRecords() collapses that to one call, returning an array of { primaryKey, key, value } records — including the index key for index-based queries.

const tx = db.transaction("notes", "readonly");
const records = await tx.objectStore("notes").getAllRecords({
  query: IDBKeyRange.lowerBound(0),
  count: 10000,
  direction: "prev",
});
// records[i] = { primaryKey, key, value }
