the code

// Spin until a flag flips. Atomics.pause() is a CPU "yield hint":
// on x86 it lowers spin-wait power use and bus contention; functionally a no-op.
function spinUntil(view) {
  while (Atomics.load(view, 0) === 0) {
    Atomics.pause();
  }
}

This page is single-threaded — we just call Atomics.pause() inside a tight loop to time the per-call overhead. The real wins land in a worker spinning on a SharedArrayBuffer.
