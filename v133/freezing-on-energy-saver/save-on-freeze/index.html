the lifecycle that energy-saver freeze adds

With Energy Saver active, Chrome 133+ now freezes hidden & silent same-origin frame groups after 5 minutes if they exceed a CPU threshold. Once frozen, your code does not run. The window to persist unsaved state is the freeze event itself.

document.addEventListener("freeze", () => { saveDraftSync(); });
document.addEventListener("resume", () => {});
addEventListener("pagehide", () => saveDraftSync());
