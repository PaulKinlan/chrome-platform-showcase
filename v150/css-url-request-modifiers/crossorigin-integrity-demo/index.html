what changes on the request

Modifier Request behavior Failure mode
cross-origin(anonymous) CORS request without cookies, client certificates, or HTTP auth credentials. Blocked when the response lacks a matching Access-Control-Allow-Origin.
cross-origin(use-credentials) CORS request with credentials included. Blocked unless the response names the page origin and sends Access-Control-Allow-Credentials: true.
integrity("sha384-...") The fetched bytes are hashed before the CSS image, font, cursor, or other resource is used. Blocked when the digest does not match, even if the HTTP response is 200 OK.
