Cross-Origin & Integrity Demo

The cross-origin() modifier sets the CORS mode for a CSS resource request. The integrity() modifier adds a Subresource Integrity check — if the fetched resource doesn't match the hash, the browser rejects it.

Checking CSS URL modifier support…

cross-origin() modifier

anonymous (default CORS)

background-image: url("https://cdn.example/img.png" cross-origin(anonymous));

Sends a CORS request without credentials. The server must respond with Access-Control-Allow-Origin. Use for public CDN resources.

use-credentials

background-image: url("https://cdn.example/img.png" cross-origin(use-credentials));

Sends cookies and credentials with the CORS request. Server must respond with Access-Control-Allow-Credentials: true and a specific origin.

integrity() modifier

Font with SRI hash

@font-face { font-family: 'Secure'; src: url("https://cdn.example/font.woff2" integrity("sha384-oqVuAfXRKap7fdgcCY5uykM6+R9GqQ8K/uxy9rx7HNQlGYl1kPzQho1wx4JwY8wC")); }

Browser verifies the fetched font matches the SHA-384 hash. If tampered or the CDN serves wrong content, the font is rejected.

Image with combined modifiers

.hero { background-image: url("https://cdn.example/hero.png" cross-origin(anonymous) integrity("sha256-47DEQpj8HBSa+/...")); }

Both modifiers together: CORS-fetch the image and verify its hash. Equivalent to <img crossorigin integrity> in HTML.

modifier reference

Modifier Values HTML equivalent Effect
cross-origin(anonymous) anonymous crossorigin="anonymous" CORS request without credentials
cross-origin(use-credentials) use-credentials crossorigin="use-credentials" CORS request with credentials
integrity("sha256-...") SRI hash string integrity="sha256-..." Verify fetched resource matches hash
referrer-policy(...) Policy values referrerpolicy="..." Control Referer header sent with request

feature detection

// CSS.supports() can detect url() modifier support
const supported = CSS.supports('background-image',
  'url("test.png" cross-origin(anonymous))');

// Or check via a style declaration
const el = document.createElement('div');
el.style.backgroundImage = 'url("test.png" cross-origin(anonymous))';
const hasModifier = el.style.backgroundImage !== '';
