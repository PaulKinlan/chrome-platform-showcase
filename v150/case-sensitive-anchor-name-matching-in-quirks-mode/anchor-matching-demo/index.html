Anchor Matching Demo

Tests fragment navigation against anchors with different case. This page uses a proper DOCTYPE, so it runs in standards mode where matching has always been case-sensitive. The demo illustrates the expected consistent behaviour across both modes in Chrome 150+.

This page is in standards mode (has <!doctype html>). In standards mode, anchor matching has always been case-sensitive. Chrome 150 extends this to quirks mode (pages without a DOCTYPE) so both modes behave consistently.
