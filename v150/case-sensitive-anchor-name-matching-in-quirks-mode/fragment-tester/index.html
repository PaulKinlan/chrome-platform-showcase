Standards mode vs quirks mode

Standards mode (has <!DOCTYPE html>)
#section → name="section" only
#Section → name="Section" only
#SECTION → name="SECTION" only
Case-sensitive since Chrome 1
Quirks mode (no DOCTYPE or old DOCTYPE)
<Chrome 150: #Section → any case variant (case-INsensitive)
Chrome 150+: #Section → name="Section" only (now case-sensitive)
