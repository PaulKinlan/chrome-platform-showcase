Styled Mirrors

The same selected option DOM is cloned into four real <selectedcontent> descendants, then styled as a toolbar chip, status bar, breadcrumb, and preview tile.

Checking selectedcontent support…
These mirrors are deliberately placed inside the select subtree. The page is probing Chrome 150's edge behavior where every descendant <selectedcontent> receives the selected option clone.
