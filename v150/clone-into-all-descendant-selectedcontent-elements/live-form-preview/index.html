A product configurator where three <selectedcontent> mirrors feed different parts of the UI simultaneously — the order summary card, a toolbar chip, and a breadcrumb trail. All three update in the same task when any dropdown changes, which is only possible with Chrome 150's fix to update all descendant selectedcontent elements, not just the first.
