Sync Inspector

Before Chrome 150, only the first <selectedcontent> in DOM order updated when a selection changed. Chrome 150 clones into all descendants. This demo shows three mirrors updating simultaneously with timestamps to confirm they fire at the same tick.

Checking selectedcontent / customizable select support…

how it works

<!-- Customizable select with multiple selectedcontent mirrors -->
<select>
  <!-- Button shows selected option anywhere via selectedcontent -->
  <button>
    <selectedcontent></selectedcontent>  <!-- Mirror A -->
  </button>

  <!-- More selectedcontent mirrors elsewhere in the select's subtree -->
  <div class="sidebar">
    <selectedcontent></selectedcontent>  <!-- Mirror B -->
  </div>
  <footer>
    <selectedcontent></selectedcontent>  <!-- Mirror C -->
  </footer>

  <option>Option 1</option>
  <option>Option 2</option>
</select>

<!-- Chrome 150: ALL three selectedcontent elements
     receive a fresh clone of the selected option simultaneously. -->
