Grid Navigation

focusgroup="horizontal vertical wrap" on a CSS grid enables full two-dimensional arrow-key navigation with zero JavaScript — ← → ↑ ↓ all work, and pressing past an edge wraps to the other side. Perfect for color pickers, emoji pickers, and game boards.

code

<!-- ONE attribute. No JavaScript. ↑ ↓ ← → all work, wraps at edges. -->
<div
  class="color-grid"
  focusgroup="horizontal vertical wrap"
  role="grid"
  aria-label="Color picker"
>
  <button class="swatch" aria-label="Red" style="background:var(--accent-rose)"></button>
  <button class="swatch" aria-label="Orange" style="background:var(--accent-amber)"></button>
  <!-- ... more swatches ... -->
</div>
