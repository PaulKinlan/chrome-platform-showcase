Tooltip over Menu

In Chrome 150, a popover=hint tooltip can remain visible while a popover=auto dropdown is open — opening the dropdown no longer force-dismisses the hint. Hover the icon buttons to show tooltips, then open a dropdown menu to see both coexist.

key rules (Chrome 150)

  • Opening a popover=auto does not force-dismiss an open popover=hint
  • Closing an auto does not cascade-close unrelated hint popovers
  • Light-dismiss (click outside) works independently for the hint and auto stacks
  • A hint sits above the auto stack in the top layer — tooltips are never hidden behind menus

code

<!-- Dropdown (auto) — light-dismisses and closes its stack -->
<button popovertarget="file-menu">File</button>
<div popover="auto" id="file-menu">…</div>

<!-- Tooltip (hint) — stays open while an auto is open -->
<button popovertarget="tip-bold" popovertargetaction="toggle">𝐁</button>
<div popover="hint" id="tip-bold">Bold (Ctrl+B)</div>

<!-- Chrome 150: hint survives the auto stack; no JS required -->
