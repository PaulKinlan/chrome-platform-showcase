Unused Preload Auditor

The headline use case. Each row below is a real <link rel="preload" as="fetch"> injected into this page. Consume some and leave others idle, then read each preload's used value — a timestamp when the resource was actually consumed, or null for wasted bandwidth.

Live used verdicts come from performance.getSpeculations(), shipping behind --enable-blink-features=SpeculationMeasurement (or --enable-experimental-web-platform-features), or the SpeculationMeasurement origin trial. Without it, the report falls back to this page's own bookkeeping of which fetches you ran — clearly labelled as such, never presented as the browser's verdict.

Report

Consume or leave preloads idle, then read the report.

// Each preloaded resource carries a `used` field:
const { preloads } = performance.getSpeculations();

for (const p of preloads) {
  if (p.used === null) {
    console.warn('Wasted preload:', p.url, '(as=' + p.as + ')');
  } else {
    console.log('Used preload:', p.url, 'at', p.used, 'ms');
  }
}
// used === null  → nothing consumed it during the page lifetime → stop preloading it
