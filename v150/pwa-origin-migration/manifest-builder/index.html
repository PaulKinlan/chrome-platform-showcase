Manifest Builder

Build the two manifest files needed for a PWA origin migration. Fill in your new origin, old origin, and app details. The builder generates the new manifest.json with scope_extensions and related_applications, plus the /.well-known/web-app-origin-association file that the old origin must serve for Chrome to accept the migration.

Both files are required: the new manifest must list the old origin in scope_extensions, and the old origin must serve a web-app-origin-association file granting the new manifest. Chrome verifies both sides before migrating an install.
