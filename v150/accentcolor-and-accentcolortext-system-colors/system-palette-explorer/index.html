My Notes
background: AccentColor; color: AccentColorText
In a real installed PWA running in Chrome 150, every occurrence of AccentColor above would resolve to the OS highlight colour automatically — no JavaScript, no CSS custom property, no explicit colour in the stylesheet. The AccentColorText keyword ensures the text on top remains readable regardless of which colour the user picked in their OS settings.

the CSS

/* Works everywhere a <color> is accepted */ .primary-btn { background: AccentColor; color: AccentColorText; /* auto-contrast, no calc() needed */ border: 2px solid AccentColor; } /* Focus rings, borders, fills — same pattern */ :focus { outline: 3px solid AccentColor; } .selected { border-left: 4px solid AccentColor; }
