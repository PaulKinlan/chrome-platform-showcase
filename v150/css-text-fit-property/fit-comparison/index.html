    value reference

    Value Grows to fill? Shrinks to fit? Typical use
    none No No (overflows) Default; text has a fixed font-size
    grow Yes No Short hero headings, banners, fill-width labels
    shrink No Yes Card titles that must fit but shouldn't enlarge
    shrink 60% No Yes, until the 60% scale clamp Labels with a readability floor

    percentage clamp

    /* Constrain how far shrink can scale */
.bounded {
  font-size: 4rem;
  text-fit: shrink 60%;  /* do not scale below 60% */
}
