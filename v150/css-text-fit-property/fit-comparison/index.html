value reference

Value Grows to fill? Shrinks to fit? Typical use
none No No (overflows) Default; text has a fixed font-size
contain Yes Yes Hero headings, banners, fill-width labels
scale-down No Yes Card titles that must fit but shouldn't enlarge

bounds with min-font-size and max-font-size

/* Constrain the range for contain */
.bounded {
  text-fit: contain;
  min-font-size: 1rem;   /* floor: never below 1rem */
  max-font-size: 4rem;   /* ceiling: never above 4rem */
  font-size: 4rem;       /* starting point for scale-down comparison */
}

/* Useful for: dashboard widgets, card titles, button labels */
