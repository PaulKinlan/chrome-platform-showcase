# Media Element Pseudo-Class Fixtures

These files are local test fixtures for the Chrome Platform Showcase media element pseudo-class
demos.

Sources:

- `flower.webm` - copied from
  `https://interactive-examples.mdn.mozilla.net/media/cc0-videos/flower.webm`
- `flower.mp4` - copied from
  `https://interactive-examples.mdn.mozilla.net/media/cc0-videos/flower.mp4`
- `t-rex-roar.mp3` - copied from
  `https://interactive-examples.mdn.mozilla.net/media/cc0-audio/t-rex-roar.mp3`

License:

The upstream MDN interactive examples repository is published under the CC0-1.0 license. The source
paths also place these fixtures under `cc0-videos` and `cc0-audio`.

Use these local files in demos instead of tiny inline data URIs so playback, seeking, buffering, and
media pseudo-class states can be tested against real media elements.
