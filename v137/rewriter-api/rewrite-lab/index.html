Rewrite lab

Pick a tone, length, and audience — the lab runs the Rewriter API four ways in parallel and renders a word-diff between original and each variant. Tracks word count and reading time so the trade is visible.

Origin trialRequires Chrome 137+ with on-device Rewriter API enabled. If unavailable the lab runs a deterministic JS stand-in so the diff/timing UI still works.
