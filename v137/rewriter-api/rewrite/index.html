Rewrite

On-device Rewriter — refactor text by tone/length/format without going to a server. Same model family as Summarizer / Translator.

The canonical shape of the API:

const r = await Rewriter.create({ tone: "more-formal" });
const out = await r.rewrite("hey, can you maybe send that?");
