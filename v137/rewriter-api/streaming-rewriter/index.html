Streaming rewriter

A live editor that re-streams every time you change a tone / length / format dial. The output rebuilds token-by-token with an abort signal — if you twist a knob mid-stream, the in-flight rewrite cancels and a new one starts. Meters track total streams, cancelled streams, and average first-token latency.

Origin trialRequires Chrome 137+ with on-device Rewriter API. Falls back to a paced simulator.
