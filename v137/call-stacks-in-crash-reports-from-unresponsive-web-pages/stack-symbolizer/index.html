Chrome 137 change: The crash report body now includes a stack array of frames when the page was unresponsive (hung). Previously, crash reports arrived with no stack information, making it impossible to identify which JavaScript was responsible for the hang. With stacks, you can feed them into your source-map / symbol toolchain to identify the specific function.
