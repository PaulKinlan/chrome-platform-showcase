IP Address Logging in NEL: Chrome 137 adds controls over how client IP addresses are handled in Network Error Logging reports. The ip_address field in the report body can be omitted or masked depending on the policy, giving site operators control over the privacy-vs-debuggability tradeoff when diagnosing network failures.
