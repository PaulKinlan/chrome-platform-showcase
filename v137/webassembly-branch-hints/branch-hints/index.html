Branch Hints

Wasm gains likely/unlikely branch hints. Compilers can produce code that lays out the hot path inline, improving CPU branch prediction.

The canonical shape of the API:

// Wasm modules can now annotate branches with
// likely/unlikely hints. Engines use them to
// improve code layout and reduce branch mispredicts.
