Ed25519

Ed25519 signature / key generation in Web Crypto. Modern curve, small keys, fast operations — usable for JWT signing, content signatures, mutual auth.

The canonical shape of the API:

const kp = await crypto.subtle.generateKey({ name: "Ed25519" }, true, ["sign", "verify"]);
const sig = await crypto.subtle.sign("Ed25519", kp.privateKey, data);
