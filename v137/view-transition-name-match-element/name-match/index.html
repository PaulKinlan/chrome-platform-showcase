view-transition-name: match-element

view-transition-name: match-element lets the browser auto-assign stable transition names per element. No more hand-managed name spaces.

The canonical shape of the API:

.card { view-transition-name: match-element; }
/* Browser auto-generates stable names per-element
   based on its identity, not the author. */
