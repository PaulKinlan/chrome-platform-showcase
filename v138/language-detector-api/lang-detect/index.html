Language Detect

On-device language detector. Runs in the same model family as Summarizer/Translator/Prompt; no network round-trip, no privacy export.

The canonical shape of the API:

const detector = await LanguageDetector.create();
const results = await detector.detect("Bore da, Cymru");
console.log(results[0].detectedLanguage);
