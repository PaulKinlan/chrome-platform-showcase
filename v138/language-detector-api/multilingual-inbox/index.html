Multilingual Inbox

An email inbox that detects the language of each message on-device using the Language Detector API. Select a message to see the ranked confidence breakdown, or compose new text and detect its language instantly — no server round-trip, no privacy exposure.

Compose — detect your own text

Privacy by design: The Language Detector API runs the model entirely on-device. Email content, message bodies, and composed text never leave the browser. This is the key difference from sending snippets to a translation service API — detection works offline and never touches a server, making it safe to use on confidential or sensitive communication.
