Shared Cache

Chrome experimentally shares cached copies of widely-loaded resources across origins, gated on strict integrity matching. Big bandwidth win for common library URLs.

The canonical shape of the API:

# Chrome can now share a single cached copy of
# extremely-popular resources (e.g. major CDN versions
# of well-known libraries) across all origins,
# reducing bandwidth and improving load times.
