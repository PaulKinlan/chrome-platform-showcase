drawElement() — HTML in a canvas

Edit the HTML on the left. Click "Draw to canvas" and the browser rasterises the live DOM into a 2D canvas via the new context.drawElement(). Rotate, scale, and composite — proper HTML rendering inside canvas.

the code

const ctx = canvas.getContext("2d");
ctx.translate(canvas.width/2, canvas.height/2);
ctx.rotate(rad);
ctx.scale(s, s);
ctx.translate(-160, -110);
ctx.drawElement(document.getElementById("src"), 0, 0);
/* drawElement rasterises a real Element subtree into the 2D
   context — text shaping, CSS, form controls, the lot. */
