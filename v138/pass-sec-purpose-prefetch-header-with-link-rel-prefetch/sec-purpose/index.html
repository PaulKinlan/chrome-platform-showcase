Sec-Purpose Header

<link rel=prefetch> now sends Sec-Purpose: prefetch. Servers can detect speculative fetches and tune their response.

The canonical shape of the API:

<link rel="prefetch" href="/next">
# Chrome now sends:
#   Sec-Purpose: prefetch
# servers can branch on this to serve a lighter response.
