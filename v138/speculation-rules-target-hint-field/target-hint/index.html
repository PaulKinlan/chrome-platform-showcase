target_hint

Speculation rules can hint at the navigation target (e.g. _blank). Lets the browser pre-allocate the right top-level context for the eventual click.

The canonical shape of the API:

{ "prerender": [{
  "urls": ["/article"],
  "target_hint": "_blank"
}] }
