Attribute Escape

Outer HTML serialisation now escapes < and > inside attribute values. Closes a class of XSS where outerHTML round-trips produced different DOMs.

The canonical shape of the API:

/* element.outerHTML now escapes < and > inside
   attribute values to &lt;/&gt; — matching the
   serialization spec. Previously they would pass
   through, enabling parser-confusion attacks. */
