Eagerness comparison

Speculation rules has four eagerness levels. Chrome 138 improves moderate on mobile — hover is less useful on touch screens, so moderate now also arms on pointerdown. Simulate user events below and watch which levels arm in time.

Level Desktop trigger Mobile trigger (138+) Typical delay Typical hit rate
conservative pointerdown + pause pointerdown + pause ~100–200 ms High — user has committed
moderate mouseover (hover) pointerdown (138+) ~200–400 ms Medium — hover intent
eager link added to DOM link added to DOM ~50–100 ms Lower — speculative
immediate as soon as possible as soon as possible < 50 ms Lowest — unconditional
