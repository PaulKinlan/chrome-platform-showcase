progress(value from start to end)

Slide a custom value through a range. progress() normalises it to 0…1, then the same scalar drives both a width and a transform — no JS, no second slider, no separate calc tree per consumer.

--val50 progress(--val from start to end)0.5 computed bar width50% computed ball scale1.25

the code

.stage {
  --val: 50;
  --p: progress(var(--val) from 0 to 100);
}
.bar  { width: calc(100% * var(--p)); }
.ball { transform: scale(calc(0.5 + var(--p) * 1.5)); }
