Spellcheck Overlay

Custom spell-check and grammar UIs require drawing underlines precisely beneath specific words in a text field. Before OpaqueRange, this was impossible without duplicating the textarea with a mirrored shadow div. createValueRange(start, end) returns the pixel bounding box of any word span directly, so you can draw on a canvas overlay with zero DOM cloning.

OpaqueRange not available. Chrome 149+ required for createValueRange(). The demo is shown using a canvas fallback that approximates character positions — in Chrome 149+ the coordinates come directly from the browser's text layout engine.
