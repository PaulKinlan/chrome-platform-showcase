← Intl.Locale.prototype.variants

v149 · Intl · Locale

Variant Tag Explorer

BCP 47 variants capture fine-grained language conventions — orthographic reforms, historical spellings, script variants. Intl.Locale.prototype.variants exposes them as an array so your app can branch logic on exact variant subtags.

BCP 47 variant subtags (e.g. 1901 for traditional German orthography, nedis for Natisone dialect Slovenian) appear after the region code. Chrome 149 now exposes them via locale.variants as a clean string[].

// Chrome 149
const loc = new Intl.Locale('de-DE-1901');
console.log(loc.variants);
// → ['1901']

// Read the variants
if (loc.variants.includes('1901')) {
  // use traditional German orthography rules
}

// Construct with variants option (also new in 149)
const loc2 = new Intl.Locale('de', { region: 'DE', variants: ['1996'] });
console.log(loc2.toString()); // 'de-DE-1996'
