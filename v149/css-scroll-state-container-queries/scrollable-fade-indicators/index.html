Scrollable Fade Indicators

Gradient fade overlays appear at the top and bottom of scroll containers only when content exists in that direction — queried entirely in CSS with scroll-state(scrollable: …). Scroll each box to see the indicators adapt.

/* Container wraps the scrollable element */
.scroll-box-wrap {
  container-type: scroll-state;
  container-name: scrollbox;
  position: relative;
}

/* Fades are always present in the DOM but hidden */
.fade-top, .fade-bottom {
  position: absolute;
  height: 48px;
  opacity: 0;
  transition: opacity 0.25s;
}
.fade-top {
  top: 0; left: 0; right: 0;
  background: linear-gradient(to bottom, white, transparent);
}
.fade-bottom {
  bottom: 0; left: 0; right: 0;
  background: linear-gradient(to top, white, transparent);
}

/* Only show fade when there IS content in that direction */
@container scrollbox scroll-state(scrollable: top) {
  .fade-top { opacity: 1; }
}
@container scrollbox scroll-state(scrollable: bottom) {
  .fade-bottom { opacity: 1; }
}
