MIME Type Filter

When an Android IME inserts media, the beforeinput event fires with inputType === "insertFromPaste" and a dataTransfer carrying the MIME type. Toggle which types your editor accepts and use the simulate buttons to replay the decision — the filter calls event.preventDefault() on rejected types.

The filter logic
// Accepted types list — built from checkboxes above const acceptedTypes = new Set(['image/gif', 'image/png', 'image/jpeg']); editor.addEventListener('beforeinput', (event) => { if (event.inputType !== 'insertFromPaste') return; const dt = event.dataTransfer; if (!dt) return; // Check each file's MIME type for (const file of dt.files) { if (!acceptedTypes.has(file.type)) { event.preventDefault(); // Block the insertion console.warn('Blocked MIME type:', file.type); return; } } // All types accepted — allow the default insertion to proceed });
