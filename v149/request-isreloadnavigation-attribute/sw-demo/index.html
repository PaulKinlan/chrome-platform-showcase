Service Worker Demo

This page registers a service worker that logs request.isReloadNavigation for every navigation request. Navigate to the page, then reload (Cmd/Ctrl+R) to see the flag toggle between false and true.

Chrome 149 required to see isReloadNavigation as true. On older browsers the attribute is undefined and the SW logs undefined. The service worker logs are routed back to this page via postMessage.

service worker code

// sw.js — installed by this page
self.addEventListener('fetch', event => {
  const req = event.request;

  if (req.mode === 'navigate') {
    const isReload = req.isReloadNavigation; // Chrome 149+

    // Notify the page so we can display it
    self.clients.matchAll().then(clients => {
      clients.forEach(c => c.postMessage({
        type: 'nav',
        url: req.url,
        isReload,
        cache: req.cache
      }));
    });
  }

  // Pass all requests through to network
  event.respondWith(fetch(req));
});
