Focus Reveal

Click any truncated cell — it reveals its full content automatically in Chrome 149. Tab to navigate. The ellipsis returns on blur. No JavaScript. The browser does it.

Chrome 149: The automatic clip-on-focus behavior is new in Chrome 149. In other browsers, clicking a cell may or may not reveal the content depending on existing input element behavior. The contenteditable reveal specifically requires Chrome 149+.

Try it: Click or tab into any cell to reveal the full text. Narrow cells show the value truncated with … — on focus, Chrome scrolls to reveal it. Press Tab to move between cells.

CSS — that's all it takes

/* The full CSS — Chrome 149 handles the reveal */
.cell {
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;

  /* Focusable — contenteditable or tabindex */
  outline: none;
}

/* Highlight on focus (optional) */
.cell:focus {
  background: #fffde7;
  outline: 2px solid blue;
}

/* Chrome 149 automatically removes the ellipsis on focus.
   No JavaScript needed to reveal the hidden text. */

Before Chrome 149, you needed JavaScript to handle this: listen for focus, set text-overflow: clip (or overflow: auto), listen for blur, restore the ellipsis. Chrome 149 makes this a browser primitive.
