shape-outside: path() and shape()

The path() and shape() functions — already supported in clip-path — now work in shape-outside. Text can wrap around arbitrary curved paths, not just polygons and ellipses.

why it shipped

CSS Shapes (the shape-outside property) shipped in Chrome 37 in 2014 with circle(), ellipse(), polygon(), and inset(). Polygon was the most powerful — but it only supports straight edges, so smooth curves required approximating them with many points. Meanwhile, clip-path gained path() in Chrome 88 and shape() in Chrome 137, giving it SVG-level curve primitives (C, Q, A, arc, curve). The disconnect was jarring: you could clip an element to a perfect teardrop curve but the text exclusion region was still a polygon approximation. Chrome 149 closes the gap — the same path() and shape() functions now work in shape-outside, letting visual and exclusion shapes match exactly.
