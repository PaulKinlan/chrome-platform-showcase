Text Wrap Studio

Pick a shape. Pick a side. Text wraps around the curved shape-outside exclusion region. Toggle between polygon(), path(), and shape() to see the improvement in edge quality.

Chrome 149: shape-outside: path() and shape-outside: shape() are new in Chrome 149. The polygon() tab works in Chrome 37+; path() and shape() require Chrome 149+. Check this page in an older browser to see the difference.
Shape: Function: Float:

The path() and shape() functions arrived in clip-path years before they came to shape-outside, creating a mismatch between the visual shape of an element and the region text avoided. An image could be clipped to a perfect teardrop curve, but the text exclusion was still a many-sided polygon approximation — the mismatch was visible at every curve. Chrome 149 closes the gap by lifting the same path() and shape() functions into shape-outside. Now the geometry is truly shared: clip the element to a wave, and the text wraps around the same wave. No approximation. No facets. The CSS Shapes specification was written with this in mind — it simply took browser implementations time to catch up.
