Caret Navigation Demo

Tab into the fields below, then use ← → and End to navigate through the full text. In Chrome 149, the ellipsis unclips so you can always see where the caret is — even in the part of the text that was hidden.

Chrome 149 required. On earlier browsers, arrow keys move the caret into the hidden region — the text stays clipped and you lose track of the cursor position. This demo highlights the improvement.

code

/* Ellipsis displayed at rest — auto-unclips on focus in Chrome 149 */
input[type="text"] {
  overflow: hidden;
  white-space: nowrap;
  text-overflow: ellipsis;
}

/* No JavaScript or :focus override needed —
   Chrome 149 handles the clip/unclip transition automatically */
