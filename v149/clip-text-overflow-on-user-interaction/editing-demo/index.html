Editing Demo

Click into the editable fields below. In Chrome 149 the ellipsis disappears and the full text becomes visible and editable. Click away to restore the clipped display.

Chrome 149 required for the automatic unclip behaviour. On earlier browsers the ellipsis remains even while you edit inside the element, making text beyond the ellipsis invisible.

What to observe

1. The initial state shows clipped text with …
2. Click the editable field — ellipsis disappears, full text visible
3. You can now navigate, select, and edit the hidden portion
4. Click elsewhere — ellipsis returns

code

/* No JavaScript needed — Chrome 149 handles this automatically */
.editable-field {
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;

  /* contenteditable, input, or textarea — all work */
}
