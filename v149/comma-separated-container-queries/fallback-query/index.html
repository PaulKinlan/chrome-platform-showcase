Fallback Query

Drag the bottom-right corner of the dashed box to resize it. The .card inside lights up if any one of the comma-separated queries matches.

I am a card

the rule

@container card-area (width > 600px), card-area (max-aspect-ratio: 1) {
  .card {
    background: color-mix(in srgb, blue 16%, paper);
    font-family: display;
    font-size: 1.4rem;
  }
}

the API

const rule = document.styleSheets[0].cssRules[0];   // the @container rule
console.log(rule.conditions);                         // ["card-area (width > 600px)",
                                                       //  "card-area (max-aspect-ratio: 1)"]
