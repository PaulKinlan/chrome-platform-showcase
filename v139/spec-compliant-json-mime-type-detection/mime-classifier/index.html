MIME Classifier

Chrome now matches the HTTP spec for JSON MIME detection — any +json suffix triggers JSON handling. Old whitelist-of-a-handful approach gone.

The canonical shape of the API:

// MIME types now recognised as JSON:
// application/json, application/*+json,
// text/json (legacy), application/manifest+json,
// application/ld+json, application/feed+json, …
