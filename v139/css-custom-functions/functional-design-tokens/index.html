Functional Design Tokens

Replace a brittle table of --space-1, --space-2, --space-3… with a single @function --space(--step). Pick a step on the slider, watch every card breathe at once. Type tier, radius scale, and contrast tone work the same way.

CSS Custom Functions (@function) is behind chrome://flags/#experimental-web-platform-features in Chrome 139. This page sets the per-card variables anyway so you can see the effect even without the flag (falling back to ordinary calc() chains); the snippets are what changes.
