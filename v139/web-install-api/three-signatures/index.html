The Three Signatures

navigator.install() ships in v139 with three call shapes: zero args (install self), one arg (install another app by URL), two args (install with manifest ID). One card per shape, each runnable.

Origin Trial: enable chrome://flags/#web-install-api or join the trial.

navigator.install()

// 0 args — install this site
await navigator.install();

Same-origin self-install. Replaces the legacy beforeinstallprompt dance.

navigator.install(url)

// 1 arg — install another site by URL
await navigator.install('https://example.com/');
idle

navigator.install(url, id)

// 2 args — install with explicit manifest id
await navigator.install(
  'https://example.com/start',
  'https://example.com/?app=v2'
);
