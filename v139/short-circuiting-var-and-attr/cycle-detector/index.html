var() Cycle Detector

Type CSS that defines a fallback chain. The v138 engine traverses every branch looking for cycles before deciding the value — so a cycle hidden in an unused fallback poisons the whole declaration. v139 short-circuits: if the primary path resolves, fallbacks aren't evaluated.
