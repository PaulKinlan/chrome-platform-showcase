Implementation Note

Windows builds of Chromium now randomise outbound TCP source ports, matching macOS/Linux. Pure security-hardening; no API change.

The canonical shape of the API:

# Chromium switches from sequential to
# randomized ephemeral TCP port allocation on Windows.
# No web-facing API change. Mitigates source-port
# prediction attacks against long-lived flows
# (NAT rebinding, off-path RST).
