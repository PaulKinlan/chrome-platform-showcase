Depth Sensing Tips

Depth-sensing data is delivered with lower latency in v139. AR apps using depth occlusion or scene understanding get fresher samples each frame.

The canonical shape of the API:

session.requestReferenceSpace("local-floor").then(refSpace => {
  session.requestAnimationFrame(function onFrame(t, frame) {
    const view = frame.getViewerPose(refSpace).views[0];
    const depth = frame.getDepthInformation(view);
    // depth.data now arrives with lower latency in v139
  });
});
