Release Note

macOS 11 (Big Sur) drops off the supported list. Users on it get a final Chrome 139 build and then no further updates.

The canonical shape of the API:

# Chrome 139 is the last version that runs on macOS 11.
# Sites: no action required — Chrome will continue
# to function, but no further updates will reach
# affected users.
