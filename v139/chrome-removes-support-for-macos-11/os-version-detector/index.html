OS Version Compatibility Detector

Chrome 139 is the last release to run on macOS 11 Big Sur. This demo reads navigator.userAgentData.getHighEntropyValues to determine the actual OS version and tell the user exactly where they stand on the support timeline.

Press “Detect my OS” to read platform version.

What should you do on your site?

Detect server-side via Sec-CH-UA-Platform-Version. Serve a targeted banner to macOS 11 users before Chrome 139 stops updating:

// Opt in to platform version hint
res.setHeader("Accept-CH", "Sec-CH-UA-Platform-Version");
res.setHeader("Critical-CH", "Sec-CH-UA-Platform-Version");

// In your handler
const platform = req.get("sec-ch-ua-platform")?.replace(/"/g, "");
const ver = req.get("sec-ch-ua-platform-version")?.replace(/"/g, "");

if (platform === "macOS" && parseInt(ver, 10) <= 11) {
  // Show "Chrome will stop updating on this OS" banner
  res.locals.showMacOs11Warning = true;
}
