routing rules

    
    
      
  • Before v139: crash reports always went to default, mixed with deprecation, intervention, CSP, and document-policy reports.
    • 
      
  • v139+: declare a crash-reporting endpoint and Chrome routes only crash reports there. If exclude-default is checked, crashes are also not mirrored to the default endpoint — your crash intake pipeline now has a clean stream.
    • 
      
  • Any report type without a matching named endpoint still falls through to default.
