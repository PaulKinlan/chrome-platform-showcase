WebGL Fallback Probe

SwiftShader (the software WebGL renderer) is gone. Devices with no GPU acceleration now get null from getContext('webgl'). Sites must branch on that. Probe your context and see the recommended pattern.

Click "Probe" to feature-detect.

const canvas = document.createElement("canvas");
const gl = canvas.getContext("webgl") || canvas.getContext("experimental-webgl");

if (gl) {
  startWebGLDemo(gl);
} else {
  // v139+: no SwiftShader fallback. Use 2D / SVG / static image.
  fallback2D();
}
