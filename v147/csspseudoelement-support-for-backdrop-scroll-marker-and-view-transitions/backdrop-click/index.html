Backdrop Click

Open the dialog, click the dimmed backdrop. The click handler attached directly to the ::backdrop pseudo fires — no hit-test against the dialog rect needed.

Click anywhere outside this dialog (i.e. on the backdrop). The ::backdrop click handler will fire and close me.

the code

const dlg = document.getElementById("dlg");
dlg.showModal();
const backdrop = dlg.pseudo("::backdrop");
backdrop.addEventListener("click", () => dlg.close());
