Before / after: Chrome 147 expanded range

OLD Chrome <147 — max 8 GB cap
NEW Chrome 147+ — up to 32 GB (desktop)

On Android, the new range is 2, 4, or 8 GB. On desktop (Windows/macOS/Linux/ChromeOS), the range expands to 2, 4, 8, 16, 32 GB. The old low-end buckets (0.25, 0.5, 1 GB) are removed — devices with those physical RAM amounts report 2 GB in Chrome 147+.
