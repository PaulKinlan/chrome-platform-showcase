# Requirements:
# 1. Both origins must share the same eTLD+1 (example.com)
# 2. Both origins must be HTTPS
# 3. A valid service worker must be registered at each origin
# 4. The manifest on the new origin must include origin_migration
# 5. The old scope must be within the old origin's scope
