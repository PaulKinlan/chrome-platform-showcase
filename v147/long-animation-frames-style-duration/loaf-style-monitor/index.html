LoAF Style Duration Monitor

Trigger style, forced-style, and layout workloads and inspect the latest PerformanceLongAnimationFrameTiming entry. Forced split fields are read from entry.scripts[], while unsupported browsers still show the measured workload duration.

When the origin trial is inactive, Chrome can still capture LoAF entries but will usually expose only the combined style/layout timing window and forcedStyleAndLayoutDuration. This monitor keeps those fallback fields visible.
