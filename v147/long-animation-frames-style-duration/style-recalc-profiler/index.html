Deliberately trigger style recalculations of varying complexity, then read styleDuration and layoutDuration from the resulting long-animation-frame PerformanceObserver entries. See exactly how much of each slow frame was style vs layout.

Origin trial feature. styleDuration / layoutDuration on LoAF entries require Chrome 147+ with the origin trial flag or a local override. The profiler runs either way; zero values mean the fields are not yet available.
