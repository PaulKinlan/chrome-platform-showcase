Nested focusgroup lab

Real toolbars nest. Format bar holds a font-family menu that itself holds a weight submenu. The focusgroup attribute composes — outer arrow keys traverse the top-level row, the extend token escapes a child into the parent's keymap, and no-memory resets the active item on exit. Toggle each and watch the keymap change.

outer toolbar — focusgroup="horizontal wrap"

inner submenu — focusgroup="horizontal extend"

key events

Behaviour map

  • focusgroup="horizontal" — left/right cycle children; up/down do nothing.
  • focusgroup="horizontal extend" — inner row's arrow keys "punch through" so the parent's left/right also traverse the inner children.
  • focusgroup="… no-memory" — leaving the group resets to the first item next time you enter.
  • focusgroup="… wrap" — right-from-last lands on first; otherwise focus stops at the edge.
<div focusgroup="horizontal wrap">
  <button>B</button>
  <button>I</button>
  <div focusgroup="horizontal extend">   <!-- inner: same axis, escapes upward -->
    <button>100</button>
    <button>400</button>
    <button>700</button>
  </div>
  <button>align</button>
</div>
