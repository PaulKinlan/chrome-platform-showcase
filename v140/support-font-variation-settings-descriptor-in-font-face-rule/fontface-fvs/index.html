@font-face FVS

Bake font-variation-settings into @font-face. The variation flags apply automatically wherever the family is used.

The canonical shape of the API:

@font-face {
  font-family: "MyFont";
  src: url("var.woff2");
  font-variation-settings: "wght" 600, "opsz" 24;
}
