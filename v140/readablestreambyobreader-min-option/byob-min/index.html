BYOB min

BYOB readers gain a min option — the read resolves only when at least min bytes are filled. Reduces the read-loop tax for sites that want bigger chunks at the cost of latency.

The canonical shape of the API:

const reader = stream.getReader({ mode: "byob" });
const { value } = await reader.read(new Uint8Array(4096), { min: 1024 });
// Resolves when at least 1024 bytes are available.
