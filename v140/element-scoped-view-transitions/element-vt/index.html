Element-scoped VT

View Transitions can now be scoped to a single element subtree. Component animations no longer have to capture the whole page.

The canonical shape of the API:

card.startViewTransition({ update() { updateCard(); } });
// Captures and animates a single element subtree
// instead of the entire document.
