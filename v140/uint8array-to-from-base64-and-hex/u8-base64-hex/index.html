Uint8Array <-> Base64/Hex

First-class base64 and hex methods on Uint8Array. Stop importing yet another encoder; the platform owns it now.

The canonical shape of the API:

const enc = bytes.toBase64();
const dec = Uint8Array.fromBase64(enc);
const hex = bytes.toHex();
const bin = Uint8Array.fromHex(hex);
