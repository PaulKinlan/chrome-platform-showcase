Segmented Control

A segmented control is just a TOC that horizontally snaps. The same scroll-target-group rule that powers a TOC powers the iOS-style segment switcher — scroll-snap takes care of the panel travel, :target-current takes care of the indicator.

real-world note

Before scroll-target-group the iOS-style segmented control required a scroll listener that hand-tagged the current segment. Worse, the indicator and the actual scroll position would desync if the user dragged mid-animation. Push it to the engine and you eliminate the desync class entirely.
