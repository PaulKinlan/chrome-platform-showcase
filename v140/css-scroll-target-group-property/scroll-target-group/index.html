Scroll Target Group

Jump between panels. Toggle the group highlighting. With scroll-target-group: highlight on the parent, a fragment target outlines the whole panel rather than the anchor element.

Mode: highlight — clicking a link outlines the entire panel.

.scroller { scroll-target-group: highlight; }
.panel:target { /* the whole panel is treated as the target */ }
