Error.isError

Error.isError(value) is the missing first-class brand check for thrown things. Unlike x instanceof Error it works across realms (iframes, workers, vm contexts) and won't be fooled by prototype tricks. The table below feeds a bunch of inputs through both checks so you can see where they diverge.

The code

Error.isError(new Error("x"));         // true
Error.isError(new TypeError("y"));     // true
Error.isError({ name: "Error" });      // false
Error.isError(crossRealmError);        // true   (instanceof would be false)
