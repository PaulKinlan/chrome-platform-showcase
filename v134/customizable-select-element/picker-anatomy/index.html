which pseudo-element does what

    
    
      
  • select { appearance: base-select; } — opts the trigger into being styleable as a real element instead of a black-box widget.
    • 
      
  • ::picker(select) — the floating menu container. Width, border, shadow, background.
    • 
      
  • ::picker-icon — the chevron / arrow on the trigger; can be replaced with any glyph.
    • 
      
  • ::checkmark — the selected-item indicator inside the picker; not used here but available for custom marks.
    • 
      
  • <selectedcontent> inside <button> — what the closed trigger displays, kept in sync with the selected option.
