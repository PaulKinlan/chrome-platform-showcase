Custom Option Builder

Build a <select> with non-option content: color swatches, badges, dividers. The relaxed HTML parser in Chrome 134 accepts arbitrary elements inside <select> — the foundation for customizable <select> with rich option content.

checking relaxed select parser…
Chrome 134 relaxes the HTML parser to allow <div>, <span>, <img>, and other elements inside <select>. Previously, the parser would immediately drop these elements and only keep the text. This is the prerequisite for the Customizable Select Element (CSS Selectmenu) which renders rich option content.
