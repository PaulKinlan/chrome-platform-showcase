option reference

    
    
      
  • start — number (DOMHighResTimestamp) or a string that names a performance.mark.
    • 
      
  • duration — number, or a second mark name. Without it the entry is an instantaneous flag.
    • 
      
  • trackGroup — top-level swim lane in the Performance panel (e.g. "App", "Framework").
    • 
      
  • track — sub-lane within the group.
    • 
      
  • color — one of the named DevTools palette tokens (primary, secondary, tertiary, error, warning).
