Notification Signal Map

Type or pick a sample notification and see the signal heatmap that Chrome's on-device model fires against. Each signal category gets a heat level; the combined score determines whether the notification body is redacted and the user sees the "Show / Dismiss / Unsubscribe" UI.

On-device LLM runs on Android Chrome — classifier simulated here
Chrome 134 runs an on-device LLM over notification text before it is shown. Notifications that score above a threshold have their body hidden and replaced with a prompt offering the user control. Detection is local — no text leaves the device. This demo simulates the signal categories the model considers.
