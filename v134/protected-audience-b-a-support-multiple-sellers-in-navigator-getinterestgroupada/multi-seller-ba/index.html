Multi-Seller B&A

Bidding and Auction services can return data for multiple sellers in one call. Reduces round-trips for multi-SSP setups.

The canonical shape of the API:

const data = await navigator.getInterestGroupAdAuctionData({
  sellers: [seller1, seller2]
});
