v134 · miscellaneous

Protected Audience (B&A): Support multiple sellers in navigator.getInterestGroupAdAuctionData

Add support for navigator.getInterestGroupAdAuctionData to return the encrypted B&A request payload for multiple sellers in a single call. This allows multiple sellers sharing a single piece of on-page JavaScript to more efficiently run an auction.

concepts

  1. Multi-Seller B&A

    Bidding and Auction services can return data for multiple sellers in one call. Reduces round-trips for multi-SSP setups.

  2. Seller Batch Builder

    Compose a multi-seller getInterestGroupAdAuctionData call live. Add sellers, configure per-buyer settings, see request body, KB, and fan-out savings. Run on Android.

    Toggle pre-v134 vs v134+ flows. Slide seller count and latency, watch the diagram redraw and saved-wall-clock counter update live.
