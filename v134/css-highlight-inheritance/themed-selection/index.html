CSS Highlight Inheritance changes how selected text propagates styles

Chrome 134 · CSS Pseudo-Elements Level 4

Before this change, ::selection styles had to be re-declared at every nesting level to ensure consistent behaviour. If you defined a custom selection colour on :root, it would apply to direct text content but might not propagate to text inside nested elements like <strong>, <em>, or <code>.

This is a blockquote-style note. Before highlight inheritance, you often had to write ::selection, .note::selection { … } to keep the selection style consistent here.

The fix is elegant: instead of inheriting through the element tree (where a child element might reset the cascade), highlight pseudo-element styles now inherit through the highlight pseudo chain. This means a single ::selection { background: … } at the root is enough for the entire document.

This makes design-system theming of selection much cleaner. Change one CSS variable and the selection colour updates everywhere — in paragraphs, inside code blocks, inside callouts, anywhere on the page.

Pre-134 vs Chrome 134+

Pre-Chrome 134
::selection did not inherit through the highlight pseudo chain — it inherited through the element tree, which could be reset by nested element styles. To reliably theme selection across an entire page authors wrote something like: *, *::before, *::after { ::selection { … } } which was verbose and fragile.
Chrome 134+
One root ::selection { background: var(--sel-bg); color: var(--sel-fg); } declaration is sufficient. Update the CSS variables and the selection theme propagates through every element and every nesting level automatically — because the highlight pseudo chain carries the values down.
/* Theme variables — change these to retheme selection everywhere */
:root {
  --sel-bg: var(--text-black);
  --sel-fg: var(--bg-ivory);
}

/* ONE rule — highlight inheritance takes it everywhere */
::selection {
  background: var(--sel-bg);
  color: var(--sel-fg);
}

/* Theme switch: just override the variables */
.theme-blue    { --sel-bg: var(--accent-blue);    --sel-fg: white; }
.theme-emerald { --sel-bg: var(--accent-emerald);  --sel-fg: white; }
.theme-rose    { --sel-bg: var(--accent-rose);    --sel-fg: white; }
