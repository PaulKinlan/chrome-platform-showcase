how it works

.card-text {
  overflow: hidden;
  white-space: nowrap;
  text-overflow: ellipsis;
  transition: max-width 0.4s ease;
}

/* CSS-only: :focus-within triggers Chrome 148's clip behaviour */
.card:focus-within .card-text {
  text-overflow: clip;
  overflow-x: auto;
}

/* Chrome 148 also does this automatically on focus — no extra CSS needed */
