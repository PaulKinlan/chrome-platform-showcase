Social Card Generator

Chrome 148's ctx.drawHtmlElement() lets you render any HTML DOM node directly into a 2D canvas context. Build a social card with a title, tagline, URL, and theme color — the card is composed as a live HTML element, then painted to a <canvas> and offered for download as a PNG.

// Chrome 148: draw an HTML element to canvas const canvas = document.querySelector('canvas'); const ctx = canvas.getContext('2d'); const el = document.querySelector('#card'); // drawHtmlElement returns a Promise — resolves after painting await ctx.drawHtmlElement(el, 0, 0, canvas.width, canvas.height); // Then export as PNG const url = canvas.toDataURL('image/png'); const a = document.createElement('a'); a.href = url; a.download = 'social-card.png'; a.click();
