Axis Explorer

Control the individual variation axes of Roboto Flex. With avar2, moving one axis can cascade through linked axes — observe how Chrome 148's improved shaping renders the intermediate states.

About avar2: The font below (Roboto Flex) uses standard variable font axes. Fonts that implement the avar2 table additionally define axis-to-axis mappings — so setting a high "Display" or "Grade" value automatically drives weight, width, and optical size in coordinated ways the font designer intended. Chrome 148 fully parses and applies these mappings for the first time.
