CSS — three lines for polished underlines

a {
  text-decoration-skip-ink: all;   /* Chrome 148 — skip beneath every glyph */
  text-underline-offset: 3px;      /* Chrome 87  — push line below descenders */
  text-decoration-thickness: 1px;  /* Chrome 87  — crisp 1px line */
}
