text-decoration-skip-ink: all

Chrome 148 adds the all value for text-decoration-skip-ink, extending ink-skip behaviour to every glyph — not just those whose descenders would intersect the underline. Cleaner underlines on every typeface, no JavaScript.

concepts

  1. Value Comparison

    Three values — none, auto, all — rendered side by side on the same text. Switch fonts and underline thickness to see how each handles descenders, curves, and Latin / CJK scripts.

  2. Link Gallery

    Realistic hyperlinks across multiple typefaces and languages using all. Toggle between values to compare the visual quality of underlines in body copy, headings, and navigation.

why it shipped

The auto value for text-decoration-skip-ink has been in Chrome since version 64. It automatically skips the underline where a glyph's ink would intersect it — great for letters with descenders like g, j, p, q, y. But auto is heuristic: it only skips where the glyph actually crosses the line, which can look inconsistent across different typefaces and writing systems. The new all value instructs the browser to skip the underline beneath every glyph, producing a uniformly spaced underline regardless of whether a specific character dips below the baseline. This matches what graphic designers achieve by nudging underlines in layout tools, and it aligns with the CSS Text Decoration Level 4 specification.
