    layer conflict panel

    Three overlapping layers all declare color. The last author layer wins — unless revert-rule is applied.

    @layer base
    color: var(--accent-blue)
    loses
    @layer theme
    color: var(--accent-emerald)
    loses
    @layer overrides
    color: var(--accent-rose) (or revert-rule)
    wins (last layer)
