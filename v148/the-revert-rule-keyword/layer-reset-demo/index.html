Layer Reset Demo

revert-layer resets a property to the value it had in the previous @layer — like an undo for the cascade. This is distinct from revert (which goes all the way back to the browser UA stylesheet) and unset (which goes to the inherited or initial value). This demo shows all three in a layered design system.

Cascade keyword comparison
Three layers: base (navy) → theme (green) → override (red). Toggle a value on the button to apply revert-layer in the last layer — it reverts to the previous layer's value.
Output
The button currently has the override (red) colour from @layer override. Toggle "Apply revert-layer" to reset it back to the theme layer's green — without removing the entire override rule.

unset: resets to inherited value or CSS initial value (ignores layers).
revert: undoes all author styles — goes back to browser defaults.
revert-layer: steps back exactly one layer — keeps all other layer styles intact.
Unlayered styles (highest implicit priority)
Any rule not in a @layer beats all layered rules
@layer componentlast declared = highest priority
.btn { background: revert-layer; }
→ uses value from @layer theme
@layer theme

@layer basefirst declared = lowest priority
.btn { background: #1e3a5f; padding: 0.5rem; }
← revert in component would skip all layers and go here or to UA
Layer priority runs bottom-to-top: @layer base (lowest) → @layer theme → @layer component (highest among layers). revert-layer in component looks down to theme. revert jumps past all layers to the browser UA stylesheet.
/* Three @layer design system */ @layer base { .btn { background: navy; } } @layer theme { .btn { background: green; } } /* revert-layer in component layer → goes back to theme layer */ @layer component { .btn-reset { background: revert-layer; } /* resolves to green (from @layer theme), not navy or browser default */ } /* Compare: */ .btn-unset { background: unset; /* → transparent (initial) */ } .btn-revert { background: revert; /* → browser UA (none) */ } .btn-rl { background: revert-layer; /* → green (prev layer) */ }
