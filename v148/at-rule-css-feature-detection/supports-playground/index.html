Chrome 148 adds @supports @rule-name { } to detect CSS at-rule support. Type any at-rule name and see live whether this browser supports it, or pick from the quick-test palette covering @layer, @container, @starting-style, @scope, and more. All tests run synchronously via CSS.supports() and StyleSheet injection.
