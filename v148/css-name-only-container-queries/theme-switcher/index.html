Theme Switcher

CSS name-only container queries let you write @container theme-root style(--theme: dark) to apply styles when a custom property matches a value — no size check needed. Toggle the theme, density, and brand color below; no JavaScript touches the component's styles. The CSS custom properties on the container drive everything.

/* The container declares custom property contexts */ /* No JS needed to switch themes — just change the property */ #app { --theme: dark; --density: compact; --brand: green; container-type: style; container-name: theme-root; } /* Component responds via name-only container queries */ @container theme-root style(--theme: dark) { .card { background: #1e293b; color: #f1f5f9; } } @container theme-root style(--density: compact) { .card { padding: 0.4rem; font-size: 0.8em; } } @container theme-root style(--brand: green) { .badge { background: #f0fdf4; color: #065f46; } }
