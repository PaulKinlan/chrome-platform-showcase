Why not use revert?

/* revert — strips ALL author styles, falls back to UA sheet */
.button { background: revert; }   /* → browser's default button appearance */

/* revert-rule — strips only @layer contributions */
.button { background: revert-rule; }  /* → non-layered author styles, then UA */

/* The difference matters in design systems:
   revert-rule lets consumers escape a layer while
   keeping non-layered author overrides intact. */
