      Spec behavior checklist
    

    

      

        
          

            Behavior
            Safe methods
            Unsafe methods
            Legacy innerHTML
          

        
        
          

            Sanitizer option
            Required by this demo: { sanitizer }
            Optional: unchecked means unrestricted default
            No platform sanitizer hook
          

          

            Baseline XSS removal
            Safe variants apply baseline protection
            No implicit baseline; result depends on the sanitizer passed
            Scripts are inert, but attributes and URLs remain
          

          

            Insertion point
            setHTML() replaces; appendHTML() appends
            setHTMLUnsafe() replaces the target contents
            Setter replaces the target contents
          

          

            runScripts
            Not available on safe dynamic markup injection
            Available through SetHTMLUnsafeOptions
            No equivalent option
