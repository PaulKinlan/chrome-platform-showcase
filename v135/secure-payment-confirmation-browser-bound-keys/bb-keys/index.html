Two signatures, one click

Secure Payment Confirmation in Chrome 135 adds a second signature — a per-device, non-syncing key the browser holds. The simulator below generates one with WebCrypto and shows how it stacks on top of the WebAuthn assertion.

Flag Browser-bound keys are gated on chrome://flags/#spc-browser-bound-keys. The demo synthesises both signatures in-page so you can inspect the shape regardless.
