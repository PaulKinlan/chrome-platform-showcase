srcdoc client tree probe

Spin up an iframe with srcdoc and ask the parent's service worker to list all of its clients. Before Chrome 135 the srcdoc frame appeared under a synthesised about:srcdoc URL and was never controlled. From 135 it inherits the parent's controller and shows up in clients.matchAll() with the parent's URL.
