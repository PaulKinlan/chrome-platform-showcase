Encoded Frame Timestamps

Encoded RTC frames expose presentation timestamps. Pipelines doing per-frame work can correlate without reaching into the codec.

The canonical shape of the API:

transformer.transform = (frame, ctl) => {
  console.log(frame.timestamp);
  ctl.enqueue(frame);
};
