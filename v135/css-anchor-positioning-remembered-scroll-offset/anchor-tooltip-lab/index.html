CSS Anchor Positioning lets tooltips and popovers position relative to a named anchor element (anchor-name / position-anchor / anchor()). Chrome 135 fixes a bug where the browser forgot the scroll offset after history navigation, causing anchored elements to appear at the wrong position on back/forward. This lab shows the corrected behaviour.
