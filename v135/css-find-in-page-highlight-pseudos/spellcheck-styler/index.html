Open Find-in-page with Ctrl/⌘ + F and search for "chrome" or "highlight" inside the preview to see the theme.

Chrome 135 introduces highlight pseudos that finally let CSS style the find-in-page UI. For years the highlight was a hard-coded yellow box. Now the page can theme it, dim it for accessibility, or emphasise the current match with its own outline. Dark-mode pages can finally stop fighting the highlight.
