Column No-Op Probe

The ::column pseudo-element only accepts post-layout properties — those that can be applied after the browser has already placed text into columns. Layout-affecting properties like width, margin, font-size, and display are silently ignored. Click any property to inject it and see whether the columns change (it works) or stay identical (it's a no-op).
