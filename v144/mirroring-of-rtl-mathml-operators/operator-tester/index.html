Operator Tester

Render the same MathML expression in LTR and RTL contexts. Asymmetric operators — &sum;, &int;, &lim; — should mirror in RTL so Arabic / Hebrew readers see them facing the right way. Toggle the dir attribute.

the API

  <mo>&int;</mo>   <!-- now mirrored -->
  <mo>&rarr;</mo>  <!-- already mirrored pre-v144 -->
