Probe each major browser storage API, write test data, and see how actual usage compares to the total quota — the number that Chrome 144 now keeps stable. Visualise the proportion each storage type contributes.

Chrome 144 makes navigator.storage.estimate() return a predictable, stable quota that doesn't drift with disk pressure. This page probes each storage API, estimates per-type usage, and displays it as a breakdown bar against the total quota.
