Quota Stabilizer

navigator.storage.estimate() now returns a stable per-origin quota that doesn't drift with system disk pressure. Predictable budgeting for offline apps.

The canonical shape of the API:

const { quota, usage } = await navigator.storage.estimate();
console.log(quota, usage);
// v144: the reported quota no longer fluctuates
// based on system free-space. Stable estimate per origin.
