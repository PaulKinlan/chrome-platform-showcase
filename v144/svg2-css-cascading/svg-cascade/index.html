SVG Cascade

SVG2 align: CSS rules now beat SVG presentation attributes in the cascade. Hard-coded fill='red' in markup is now overridable by a sufficiently-specific CSS rule.

The canonical shape of the API:

<svg>
  <style>
    rect { fill: red; }
    rect[fill="blue"] { fill: green; }
  </style>
  <rect fill="blue" />
  <!-- Per SVG2: CSS cascade beats presentation attribute. -->
</svg>
