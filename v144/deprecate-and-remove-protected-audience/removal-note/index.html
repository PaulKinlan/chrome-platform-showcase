Removal Note

The Protected Audience API (formerly FLEDGE) joins the Privacy Sandbox deprecation. Sites using on-device ad auctions should plan a migration off the API.

The canonical shape of the API:

# Protected Audience (FLEDGE) API is deprecated
# and being removed alongside the broader Privacy
# Sandbox wind-down. Ad-tech using runAdAuction(),
# joinAdInterestGroup(), etc. should migrate.
