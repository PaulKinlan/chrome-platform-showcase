Shared Storage Note

Another Privacy Sandbox removal — Shared Storage is gone. Sites should rely on standard per-origin storage from here on.

The canonical shape of the API:

# Shared Storage API removed. Sites should use
# normal localStorage / IndexedDB (origin-scoped)
# and avoid the cross-site storage assumptions
# Shared Storage allowed.
