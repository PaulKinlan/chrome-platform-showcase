rSAFor Note

rSAFor — top-level requesting on behalf of embedded sites — is gone. The embedded site itself must call requestStorageAccess() now.

The canonical shape of the API:

# document.requestStorageAccessFor is removed in
# v144. Use document.requestStorageAccess() inside
# the third-party context that needs cookies.
