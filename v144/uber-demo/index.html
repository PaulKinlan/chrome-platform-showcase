Chrome 144 features at work

featurerole in this demo
Temporal in ECMA262 All event times are stored as Temporal.Instant values. Display in each timezone uses Temporal.ZonedDateTime — DST transitions, ISO week boundaries, and calendar-aware formatting handled automatically with no Date arithmetic.
@scroll-state scrolled The sticky planner header listens to @container app-frame scroll-state(scrolled: y) and reduces its padding and background when the event list has been scrolled. Zero JavaScript scroll listeners involved.
Scroll Triggered Animations Each event card has animation-timeline: view() with animation-range: entry 0% 70%. Cards fade and slide in as they scroll into the viewport — the animation fires once, on entry.
CSS caret-shape: block Both form inputs declare caret-shape: block — the text cursor becomes a solid block matching terminal / code-editor UX. Browsers without support fall back to the default bar caret gracefully.
clipboardchange event A clipboardchange listener fires immediately every time the system clipboard changes. The handler reads the new text, runs it through a Temporal date-parser, and surfaces a one-click "Use this time" button if the string is recognisable.

code

/* @scroll-state scrolled (Chrome 144) — sticky header compresses on scroll */
@container app-frame scroll-state(scrolled: y) {
  .app-header {
    padding-top: 0.5rem;
    background: var(--bg-stone);
  }
}

/* CSS caret-shape (Chrome 144) — block cursor in inputs */
.form-input { caret-shape: block; }

/* Scroll Triggered Animation (Chrome 144) */
.event-card {
  animation: card-enter 0.4s ease both;
  animation-timeline: view();
  animation-range: entry 0% 70%;
}

// Temporal (Chrome 144) — timezone-aware event display
const instant = Temporal.Instant.from(isoString);
const inTokyo = instant.toZonedDateTimeISO('Asia/Tokyo');
display(inTokyo.toLocaleString('en-GB', { dateStyle: 'medium', timeStyle: 'short' }));

// clipboardchange (Chrome 144) — fire on every clipboard write
document.addEventListener('clipboardchange', async () => {
  const text = await navigator.clipboard.readText();
  tryParseDate(text);
});
