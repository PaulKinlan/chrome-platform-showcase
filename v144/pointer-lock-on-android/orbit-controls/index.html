Orbit Controls

The motivating use case for shipping Pointer Lock on Android: 3D viewers and games where the cursor would otherwise hit the screen edge. Tap the arena to lock the pointer, drag indefinitely to spin the cube. Works on desktop already; v144 enables Android with a connected pointing device.

Pointer Lock API: checking…

tap the arena to lock the pointer; move to orbit; esc to release

the API

arena.addEventListener("click", () => arena.requestPointerLock());
document.addEventListener("pointerlockchange", () =>
  isLocked = (document.pointerLockElement === arena));
document.addEventListener("mousemove", e => {
  if (isLocked) { yaw += e.movementX; pitch += e.movementY; }
});
