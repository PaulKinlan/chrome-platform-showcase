Pointer Lock on Android

Click the arena to request pointer lock. Move a mouse — the red dot integrates relative movementX/Y (same code path desktop already used). On Android Chrome 144 with a pointing device attached, this now works.

the API

arena.addEventListener('click', () => arena.requestPointerLock());

document.addEventListener('pointerlockchange', () => {
  console.log('locked element:', document.pointerLockElement);
});
document.addEventListener('mousemove', (e) => {
  if (document.pointerLockElement) integrate(e.movementX, e.movementY);
});
