XXE Note

Chrome's XML parser no longer follows external entity references. Closes a long-standing XXE vector that affected DOMParser and friends.

The canonical shape of the API:

<!-- Pre-v144: certain XML APIs would follow
     external entity references, enabling XXE. -->
<!-- Post-v144: external entities ignored entirely. -->
