<usermedia> Element

A declarative <usermedia> element that requests an input device and emits a capture event with the stream. No JS for the common case of 'tap to record'.

The canonical shape of the API:

<usermedia kind="audioinput" oncapture="record(event)">
</usermedia>
<!-- Declarative replacement for getUserMedia
     for the common request-microphone case. -->
