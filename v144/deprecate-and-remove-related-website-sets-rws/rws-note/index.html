RWS Deprecation Note

Related Website Sets is being removed. Most multi-origin grouping use cases should switch to a single primary origin plus subdomains under one eTLD+1.

The canonical shape of the API:

# RWS no longer supported. document.requestStorageAccessFor
# also being removed. Sites should rely on Storage
# Access API alone within true same-origin scope.
