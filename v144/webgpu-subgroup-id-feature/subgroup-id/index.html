subgroup_id

Adds subgroup_id as a WGSL built-in — lets compute shaders coordinate within subgroups (warp/wavefront) without ambient knowledge of the device.

The canonical shape of the API:

@compute @workgroup_size(64)
fn main(@builtin(subgroup_id) sid: u32) {
  // Identify which subgroup within the workgroup
  // this invocation belongs to.
}
