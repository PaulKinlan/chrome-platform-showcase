Checking CSS anchor positioning support…
The Chrome 144 fix: Before Chrome 144, anchor() resolved positions against the element's layout box, ignoring CSS transforms. Chrome 144 makes anchor positioning transform-aware — the anchored element now tracks the visual (post-transform) position.
