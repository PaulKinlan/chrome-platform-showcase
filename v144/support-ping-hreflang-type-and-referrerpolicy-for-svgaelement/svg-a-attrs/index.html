SVG Anchor Attributes

SVG <a> now honours ping, hreflang, type, and referrerpolicy — bringing it to parity with HTML <a>.

The canonical shape of the API:

<svg>
  <a href="/next" hreflang="en" type="text/html"
     ping="/log/clicks" referrerpolicy="no-referrer">
    <rect width="100" height="40" />
  </a>
</svg>
