High-entropy Policy

Embedders can lock out high-entropy User-Agent Client Hints via Permissions Policy, mirroring the existing low-entropy controls. Tightens fingerprinting surface.

The canonical shape of the API:

Permissions-Policy: ch-ua-high-entropy-values=()
# Now blocks high-entropy UA-CH hints from being
# sent by this document and any embedded iframes
# unless they explicitly request the permission.
