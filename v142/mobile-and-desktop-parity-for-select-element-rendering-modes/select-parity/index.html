Select Parity

<select> rendering modes (drop-down vs list-box, base-appearance) now behave the same across mobile and desktop. Reduces conditional CSS for cross-platform forms.

The canonical shape of the API:

<select size="0">
  <option>One</option>
  <option>Two</option>
</select>
/* Pre-v142: mobile and desktop rendered very
   differently. Post-v142: aligned. */
