Format Tiers

WebGPU introduces tiered optional format support. Sites opt in to tier1/tier2 once for a coherent expanded format set, instead of negotiating individual format features.

The canonical shape of the API:

const device = await adapter.requestDevice({
  requiredFeatures: ["texture-formats-tier1"]
});
// Now allowed: rg32float storage, rgba32float copy_src/dst, …
