Input Mode Matrix

The point of interestfor is that one attribute responds to every way a user might signal interest — mouse hover, keyboard focus, touch long-press, screen-reader virtual cursor, even pointer-pen hover. Each row below shows whether your current input device has surfaced an interest event yet. Touch each target on whatever device you brought.
