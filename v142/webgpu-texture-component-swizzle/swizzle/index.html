Swizzle

WebGPU texture views can remap channels at sample time. Saves a copy when an asset's channel ordering doesn't match what the shader expects.

The canonical shape of the API:

const view = texture.createView({
  componentSwizzle: { r: "g", g: "r", b: "b", a: "a" }
});
