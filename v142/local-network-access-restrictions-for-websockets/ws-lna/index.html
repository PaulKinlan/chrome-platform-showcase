WebSocket LNA

Local Network Access permission now also gates WebSocket connections to private/loopback addresses. Closes the parallel attack surface.

The canonical shape of the API:

const ws = new WebSocket("ws://192.168.1.1/");
// Triggers Local Network Access permission prompt
// before opening the socket.
