Multicast Receiver

Direct Sockets gain multicast send/receive. Required for industrial / IoT discovery protocols (mDNS, SSDP, custom factory floor).

The canonical shape of the API:

const socket = new UDPSocket({
  remoteAddress: "224.0.0.1",
  remotePort: 5353,
  multicast: { interfaces: ["wlan0"] }
});
