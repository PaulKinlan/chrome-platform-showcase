Sticky Activation

User activation carries across same-origin renderer-initiated navigations. Sites split across multiple URLs no longer lose activation-gated capabilities at every navigation.

The canonical shape of the API:

// Click on /page-a, navigate to /page-b within same origin,
// page-b now retains the user-activation bit and can
// programmatically focus/play media/etc. without
// asking the user to click again.
