Strict JSON MIME

JSON modules require strictly spec-conformant +json MIME tokens. Catches sites that serve malformed Content-Type headers; brings Chrome in line with V8's module loader spec text.

The canonical shape of the API:

// import config from "./config.json" with { type: "json" };
// Server must respond with a strictly valid +json
// MIME (no malformed parameters, exact suffix match).
