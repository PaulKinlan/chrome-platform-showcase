primitive_index

WGSL gains a primitive_index built-in in fragment shaders. Lets shaders identify which triangle they're shading without packing the index into a per-vertex attribute.

The canonical shape of the API:

@fragment fn main(@builtin(primitive_index) i: u32) -> @location(0) vec4f {
  // Pick a colour per triangle, no per-vertex attribute.
  return vec4f(f32(i & 7u) / 7.0, 0.0, 0.0, 1.0);
}
