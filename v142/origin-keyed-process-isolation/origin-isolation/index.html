Origin Isolation

Origin-Agent-Cluster header now reliably gives the origin its own renderer process. Stronger Spectre containment, predictable behaviour for sites that opt in.

The canonical shape of the API:

Origin-Agent-Cluster: ?1
# Tells Chrome to put this origin in its own renderer
# process, separate from siblings on the same eTLD+1.
# Better Spectre mitigation; small memory cost.
