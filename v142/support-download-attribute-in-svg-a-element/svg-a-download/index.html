SVG <a> download

SVG <a> elements now honour the download attribute the same way HTML <a> does. Final piece of the SVG-anchor attribute parity work.

The canonical shape of the API:

<svg>
  <a href="report.pdf" download="2026-report.pdf">
    <rect width="200" height="60" />
  </a>
</svg>
