Range Container Queries

Style container queries (and the new if() function) accept range syntax like @media. Lets authors gate styles on numeric custom-property thresholds without nested conditionals.

The canonical shape of the API:

@container style(--brand-density >= 2) {
  .card { padding: 2rem; }
}
.title {
  font-size: if(style(--scale > 1.4): 2rem; else: 1rem);
}
