LNA Prompt

Public-origin requests targeting private/loopback IPs go through a Local Network Access permission prompt. Mitigates the classic 'router CSRF' attack class.

The canonical shape of the API:

// fetch("http://192.168.1.1/") from a public origin
// triggers a permission prompt before the request leaves
// the browser. Designed to mitigate CSRF-style attacks
// against local routers and dev servers.
