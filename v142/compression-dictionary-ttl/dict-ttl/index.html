Dictionary TTL

Compression Dictionary Transport gets explicit TTL semantics — servers control how long the browser keeps a dictionary cached. Stops orphan dictionaries from lingering.

The canonical shape of the API:

# Server sets:
#   Use-As-Dictionary: match="/page-*", ttl=86400
# Browser drops the cached dictionary after the TTL
# instead of holding it indefinitely.
