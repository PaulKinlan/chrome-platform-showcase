Active VT

document.activeViewTransition exposes the currently-running ViewTransition (or null). Lets sites coordinate competing transition triggers without their own state.

The canonical shape of the API:

if (document.activeViewTransition) {
  // A transition is in flight. Skip starting another.
  return;
}
startViewTransition(...);
