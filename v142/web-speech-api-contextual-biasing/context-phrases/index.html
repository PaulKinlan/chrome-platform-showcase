Context Phrases

Speech recognition accepts a contextPhrases hint — strings the engine should bias towards. Massive accuracy bump for domain-specific vocabularies (place names, jargon, brand names).

The canonical shape of the API:

const sr = new SpeechRecognition();
sr.contextPhrases = ["Llanfairpwllgwyngyll", "Cwmbran"];
sr.start();
// The recogniser biases towards these phrases
// when it sees acoustically-similar input.
