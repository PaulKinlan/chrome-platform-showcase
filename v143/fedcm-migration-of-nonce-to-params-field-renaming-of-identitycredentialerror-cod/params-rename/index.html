params Rename

FedCM nonce moves out of a top-level field into a structured params object. IdentityCredentialError's code attribute is renamed to error for alignment. Migration aid for sites already using FedCM.

The canonical shape of the API:

navigator.credentials.get({
  identity: {
    providers: [{ configURL, clientId, params: { nonce: "..." } }]
  }
});
