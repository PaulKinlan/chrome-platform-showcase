Manifest Update Hints

Web App Manifest gets a way to declare that icon URLs are immutable. Lets Chrome skip eligibility checks on update, speeding up update detection.

The canonical shape of the API:

{
  "icons": [{ "src": "/icon-v3.png", "sizes": "512x512" }],
  "update_eligibility": { "icon_urls": "immutable" }
}
