WebTransport ALPN

WebTransport sessions can negotiate an application-level protocol via ALPN, the same way WebSocket does. Lets a single endpoint serve multiple protocol versions cleanly.

The canonical shape of the API:

const wt = new WebTransport("https://example.com/", {
  serverCertificateHashes: [...],
  protocols: ["my-app/1", "my-app/2"]
});
console.log(wt.protocol);  // negotiated protocol
