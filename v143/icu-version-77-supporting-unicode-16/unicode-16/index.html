Unicode 16

Chrome's ICU bumps to version 77, picking up Unicode 16. New emoji, new scripts, refreshed segmentation rules — all available through Intl, regex, and string APIs.

The canonical shape of the API:

console.log("\u{1F6DC}");  // U+1F6DC HUT (new in Unicode 16)
console.log(new Intl.Segmenter("en", { granularity: "grapheme" })
  .segment("\u{1F6DC}").next().value);
