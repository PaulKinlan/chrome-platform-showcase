JSON Account List

FedCM IdP endpoints can return structured JSON responses, simplifying both implementation and validation. Old text format still accepted during the migration window.

The canonical shape of the API:

# IdP can now respond with:
#   Content-Type: application/json
#   { "accounts": [...] }
# instead of the old text-based format.
