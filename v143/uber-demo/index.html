CSS At-Rule Inspector

Five Chrome 143 features in one tool. CSS.supports('at-rule(@layer)') probes every CSS at-rule; results are tallied with Map.prototype.getOrInsert. Hovering a row opens an anchor-positioned tooltip with @container anchored(fallback:) re-colouring the direction hint as it flips. The CSS scanner reads your code with Blob.bytes(). An install prompt uses the new Web Install API if available.

Chrome 143 features at work

featurerole in this demo
@supports at-rule() Every row in the matrix calls CSS.supports('at-rule(@media)'), CSS.supports('at-rule(@layer)'), and so on — the new Chrome 143 function inside @supports that tests whether a CSS at-rule is supported, not just a property.
CSS Anchored Fallback CQs The detail tooltip uses anchor-name on the hovered row and position-anchor on the tooltip. A position-try-fallbacks chain flips the tooltip to avoid viewport overflow. @container anchored(--tt-below) re-styles the direction hint text in the tooltip based on which fallback is active.
Map.prototype.getOrInsert Probe results are tallied in a Map<string, {count: number}> using map.getOrInsert(key, { count: 0 }).count++. This is the Chrome 143 upsert pattern: a single call either retrieves the existing entry or inserts the default — no separate has()/set() guard needed.
Blob.bytes() When you click Scan, the textarea content is wrapped in a Blob and read with blob.bytes() — the Chrome 143 method that returns a Promise<Uint8Array> directly instead of requiring the arrayBuffer() → new Uint8Array() round-trip. The byte array is decoded as UTF-8 for the at-rule count.
Web Install API The "Install Inspector" button appears only if 'install' in navigator, the Chrome 143 Web Install API capability gate. Clicking it calls navigator.install() — no user gesture capture or beforeinstallprompt hacking needed.

/* @supports at-rule() (Chrome 143) — detect CSS at-rule support */
const supported = CSS.supports('at-rule(@layer)');
const isMedia   = CSS.supports('at-rule(@media)');

// Map.prototype.getOrInsert (Chrome 143 upsert) — no has()/set() guard
const tally = new Map();
for (const rule of atRules) {
  tally.getOrInsert(rule, { count: 0 }).count++;
}

// Blob.bytes() (Chrome 143) — Uint8Array without arrayBuffer() round-trip
const blob  = new Blob([cssText], { type: 'text/css' });
const bytes = await blob.bytes();  // → Uint8Array
const text  = new TextDecoder().decode(bytes);

/* CSS Anchored Fallback Container Query (Chrome 143) */
.tooltip {
  position-anchor: --row-1;
  position-try-fallbacks: --tt-below, --tt-above, --tt-right;
}
@container anchored(--tt-below) {
  .direction-hint { color: var(--accent-blue); }
}

// Web Install API (Chrome 143)
if ('install' in navigator) {
  installBtn.onclick = () => navigator.install();
}
