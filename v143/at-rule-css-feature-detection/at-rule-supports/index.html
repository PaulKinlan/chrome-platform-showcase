@supports for at-rules

@supports gains an at-rule() function for detecting whether a given at-rule (e.g. @scope, @starting-style) is supported. Closes the gap left by the existing property/selector tests.

The canonical shape of the API:

@supports at-rule(@scope) {
  /* The browser understands @scope. */
}
