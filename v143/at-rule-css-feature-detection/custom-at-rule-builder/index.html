How it works

    
    
      
  1. The at-rule() function inside @supports is new in Chrome 143. It returns true if the browser understands the named at-rule as a valid CSS construct.
    2. 
      
  2. In JavaScript, CSS.supports("at-rule(@layer)") mirrors this — returns a boolean you can branch on.
    3. 
      
  3. For progressive enhancement, wrap declarations in @supports at-rule(@property) { … } so only browsers that parse @property apply them.
