Fallback Container Query

Container queries learn the same fallback chain mechanics as @supports. A query block applies if any one of several conditions match, including style-query branches.

The canonical shape of the API:

@container (width > 600px) or (--design-system: lite) {
  .card { padding: 2rem; }
}
