Spellcheck Fence

Authors can restrict which substrings inside an editable region get the browser's spell/grammar highlight. Helpful for code editors where parts of the value are programmatic.

The canonical shape of the API:

<textarea spellcheck="false"
         spellcheckrange="50-200">…</textarea>
