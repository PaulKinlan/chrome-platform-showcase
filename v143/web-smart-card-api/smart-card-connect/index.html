Smart Card Connect

PC/SC-style smart card access from the web. Required for many enterprise SSO flows (government ID cards, hardware tokens) that previously needed native helpers.

The canonical shape of the API:

const ctx = await navigator.smartCard.establishContext();
const readers = await ctx.listReaders();
const conn = await readers[0].connect("shared");
const apdu = new Uint8Array([0x00,0xA4,0x04,...]);
const resp = await conn.transmit(apdu);
