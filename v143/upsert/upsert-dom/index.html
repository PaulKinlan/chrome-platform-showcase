Upsert DOM

DOM Element.upsert() — a single primitive for the common 'replace this child if it exists, otherwise append' pattern. Cleaner than the conditional replaceChild/append dance.

The canonical shape of the API:

parent.upsert(newChild, oldChild);
// Replace if found, else append. One call.
