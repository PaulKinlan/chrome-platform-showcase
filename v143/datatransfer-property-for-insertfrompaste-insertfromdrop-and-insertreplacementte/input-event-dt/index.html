InputEvent.dataTransfer

InputEvent now carries a dataTransfer reference for paste / drop / replacement events. Editors get a typed handle on the inserted payload without listening on multiple event types.

The canonical shape of the API:

editor.addEventListener("beforeinput", (e) => {
  if (e.inputType === "insertFromPaste") {
    for (const item of e.dataTransfer.items) {
      console.log(item.kind, item.type);
    }
  }
});
