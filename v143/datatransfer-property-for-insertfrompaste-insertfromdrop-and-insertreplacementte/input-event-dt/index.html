InputEvent.dataTransfer

Paste, drop, or use spell-check replacement into the editor below. The log shows the live beforeinput event with its inputType and the items inside event.dataTransfer. Pre-v143, dataTransfer was always null on these events.

Try: Cmd/Ctrl+V some HTML, drag a file or image in, or right-click a misspelled word and pick a suggestion.

the call

editor.addEventListener("beforeinput", (e) => {
  if (e.inputType === "insertFromPaste" ||
      e.inputType === "insertFromDrop"  ||
      e.inputType === "insertReplacementText") {
    // v143: e.dataTransfer is no longer null
    for (const item of e.dataTransfer.items) {
      console.log(item.kind, item.type);
    }
  }
});
