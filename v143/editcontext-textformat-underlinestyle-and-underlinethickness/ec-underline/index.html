EditContext Underline

EditContext gains underlineStyle (solid/dashed/dotted/etc) and underlineThickness fields on TextFormat. Custom editors can render IME composition underlines accurately.

The canonical shape of the API:

editContext.addEventListener("textformatupdate", (e) => {
  for (const fmt of e.textFormats) {
    console.log(fmt.underlineStyle, fmt.underlineThickness);
  }
});
