UBO Layout

An opt-in WebGPU feature that relaxes uniform buffer layout rules to std140-compatible. Big help for ports from GLSL.

The canonical shape of the API:

const device = await adapter.requestDevice({
  requiredFeatures: ["uniform-buffer-standard-layout"]
});
