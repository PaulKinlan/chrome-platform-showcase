Prefixed video FS removed

Removes the long-deprecated webkitEnterFullscreen / webkitExitFullscreen on <video>.

The canonical shape of the API:

// video.webkitEnterFullscreen() / webkitExitFullscreen()
// removed. Use the standard element.requestFullscreen().
