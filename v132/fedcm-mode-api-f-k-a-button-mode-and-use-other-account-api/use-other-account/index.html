Use Other Account: adding a second identity at the same IdP

Before 132, the FedCM sheet showed only accounts the IdP had previously surfaced to the browser. With mode: "active" + Use Other Account, the sheet adds an “use a different account” row that opens an IdP-controlled login window in the same flow — useful when a user wants to switch to a work identity at the same provider.
