Capture Links Configurator

Chrome 132 adds user navigation capturing on desktop: links to an installed PWA's origin open in the app instead of the browser. The capture_links manifest field controls how this works. Configure it below, preview the manifest JSON, and simulate which links would open in the app vs. the browser.
