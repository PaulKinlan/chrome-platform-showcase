Keyboard-focusable scrollers

Scroll containers are now keyboard-focusable by default. Long-overdue accessibility fix for arrow-key navigation.

The canonical shape of the API:

/* Scroll containers (overflow: auto/scroll) are now
   reachable with Tab and can be scrolled with arrow keys
   — without an explicit tabindex. */
