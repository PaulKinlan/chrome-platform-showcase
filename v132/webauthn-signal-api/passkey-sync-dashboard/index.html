Passkey Sync Dashboard

The WebAuthn Signal API lets relying parties report credential state back to credential providers (password managers, OS passkey stores). Use the dashboard to manage a simulated passkey list and fire the three signal calls: reportExistingCredential, reportAllAcceptedCredentials, and reportAllDeletedCredentials.
