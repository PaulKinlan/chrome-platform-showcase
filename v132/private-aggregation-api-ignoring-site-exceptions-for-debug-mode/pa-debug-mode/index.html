PA Debug Mode

Private Aggregation debug mode bypasses site exceptions. Developer overrides always work, regardless of user-level opt-outs.

The canonical shape of the API:

# Debug mode in Private Aggregation ignores site-level
# disable exceptions, so devs can always inspect what
# would have been sent.
