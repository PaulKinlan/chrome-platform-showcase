Inactive doc throw

showModal()/showPopover() on inactive documents throw InvalidStateError instead of silently no-oping. Surfaces bugs that previously slipped past.

The canonical shape of the API:

// dialog.showModal() / popover.showPopover() inside
// an inactive document (detached frame, bfcache) now
// throws InvalidStateError instead of silently failing.
