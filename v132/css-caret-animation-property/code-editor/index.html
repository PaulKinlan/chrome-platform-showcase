Code Editor

A code editor mockup with four caret modes — Normal (steady grey), Insert (green blink), Select (blue steady), and Search (red blink). Each mode sets caret-color and caret-animation on the textarea. Familiar signalling for Vim users, implemented in pure CSS.

Feature detection: checking…

Switch modes with the toolbar. In a real editor these would be triggered by keyboard shortcuts — here they toggle the data-mode attribute which CSS selects on.

/* Map Vim modes to caret styles */

[data-mode="normal"] textarea {
  caret-color: #888;
  caret-animation: none;   /* steady bar — not editing */
}

[data-mode="insert"] textarea {
  caret-color: #6bc98a;
  caret-animation: blink;  /* green blink — ready to type */
}

[data-mode="visual"] textarea {
  caret-color: #7aa2e8;
  caret-animation: none;   /* blue steady — selection mode */
}

[data-mode="search"] textarea {
  caret-color: #e87a7a;
  caret-animation: blink;  /* red blink — find mode */
}
