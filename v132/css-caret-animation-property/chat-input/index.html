Chat Input

A chat interface where the input caret changes style based on the current input mode: blink for normal text, none (steady) for code snippets, and a coloured caret for @mention mode. Context-sensitive caret signalling — zero JavaScript.

Click into the message input. Type @ to trigger mention mode (blue caret), type a backtick to trigger code mode (green steady caret). Click any conversation to switch.

/* Normal typing: classic blink */
textarea[data-mode="normal"] {
  caret-color: black;
  caret-animation: blink;
}

/* @mention mode: coloured, still blinks */
textarea[data-mode="mention"] {
  caret-color: var(--accent-blue);
  caret-animation: blink;
}

/* Code snippet mode: steady (like a terminal) */
textarea[data-mode="code"] {
  caret-color: var(--accent-emerald);
  caret-animation: none;
}
