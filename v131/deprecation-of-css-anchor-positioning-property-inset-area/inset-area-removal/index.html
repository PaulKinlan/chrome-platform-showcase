inset-area renamed to position-area

The CSS Anchor Positioning property inset-area was renamed to position-area. This demo applies the new name and lets you switch the side so you can see the live behaviour.

position-area: top
anchor

the css

/* old, deprecated */
.popover { inset-area: top; }
/* new, preferred */
.popover { position-area: top; }
