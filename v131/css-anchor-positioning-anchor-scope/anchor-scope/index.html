anchor-scope confines anchor names

Two lists both declare anchor-name: --target on every row. Hover any row to anchor the badge. Toggle anchor-scope to see what happens when names collide vs when each list owns its own scope.

A · row 2
A · row 3
anchored badge

List B

B · row 1
B · row 2
B · row 3
anchored badge

If your browser lacks anchor positioning support the badges will fall back to the top-left of each list.

the css

.list { anchor-scope: --target; }
.row { anchor-name: --target; }
.badge {
  position: absolute;
  top: anchor(--target bottom);
  left: anchor(--target right);
}
