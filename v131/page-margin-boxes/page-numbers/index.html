Chapter 2 — Adoption Living standard

Page Margin Boxes in CSS

Paged media has always needed sixteen edges. Top-left for the document title, top-center for the running header, top-right for the section name. Centered footers for "page X of Y" pulled from counters. Corner boxes for crop marks. Before v131, browsers ignored most of them.

Now @page + @top-left + @bottom-center { content: "Page " counter(page) " of " counter(pages); } just works in Chrome's print pipeline.

Use the dropdown to swap layouts. Hit "print preview" to see the boxes render in the real paged engine.
