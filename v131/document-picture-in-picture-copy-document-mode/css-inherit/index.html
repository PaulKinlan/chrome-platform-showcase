style inheritance: host theme follows the PiP

Pick a design (cyber, editorial, minimal). Open the PiP window with copied styles, then open the same card without copied styles. The v131 option makes the popout inherit the host theme instead of rendering as a bare unstyled document.

The two buttons clone the same node. Only the request option changes, so the PiP window either receives this page's <link>/<style> sheets or falls back to browser defaults.
