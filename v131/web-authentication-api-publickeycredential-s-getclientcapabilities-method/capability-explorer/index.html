PublicKeyCredential.getClientCapabilities() returns a record of WebAuthn features the client implements: conditional UI, large blob, related origin requests, payment credentials, more. Each tile below shows one capability — green if your browser exposes it, red if it doesn't, grey if untested. Use this to gate which sign-in surface you offer.
